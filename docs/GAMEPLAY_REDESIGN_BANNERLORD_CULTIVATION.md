# 灵枢笔录 · 骑砍式修仙玩法重构方案

> 日期: 2026-05-27  
> 目标: 解决“用户没有打怪欲望、社交信息冗余、长期耐心不足、数值失控”的核心问题  
> 参考方向: 《Mount & Blade II: Bannerlord》式沙盒角色扮演 + 修仙题材长期成长 + 简单上瘾游戏的一局构筑循环  
> 硬约束: 仍然要求用户上传并验证自己的 API Key,不做免 Key 试玩。

---

## 0. 结论先行

当前《灵枢笔录》的问题不是“不够多”,而是“目标不够硬”。  
玩家现在打怪的理由偏弱:

```text
打怪 -> 获得经验/材料 -> 回主城
```

这个循环在新鲜感过去后很容易疲劳。玩家不会为了“再看一段 AI 文案”持续打怪,也不会记住大量真实公司化身、人名和复杂宗门关系。

新的核心应改成:

```text
你不是普通修士,你是一个小宗门/小队的掌门。
你带着弟子和道兵在灵脉大陆上游走。
你打怪不是为了刷怪,而是为了抢资源、扩队伍、夺据点、改写势力格局。
```

一句话产品定位:

```text
修仙版骑马与砍杀:
从一名落魄执笔者起步,招募弟子,夺取灵脉,建立宗门,在五大道统与妖魔潮之间打出自己的地盘。
```

核心重构:

| 当前 | 新方案 |
|---|---|
| 地图点怪 | 世界地图跑商、接悬赏、打据点、夺灵脉 |
| 怪物列表 | 妖巢、魔寨、秘境、灵矿、商路威胁 |
| 复杂社交人名 | 5 大势力 + 12 个清晰族群 + 少量可记住的掌门 |
| 主角单人刷怪 | 主角 + 弟子 + 道兵小队 |
| 战斗只影响自己 | 战斗改变资源点、商路安全、宗门声望、地盘归属 |
| AI 文案是奖励 | AI 文案只用于大事件:首战、破境、攻城、灭门、建宗 |
| 数值膨胀 | 低数值、可读、强约束的战力模型 |

## 1. 外部调研摘要

### 1.1 骑马与砍杀为什么让人持续玩

《Mount & Blade II: Bannerlord》的 Steam 页面把它定义为策略/动作 RPG: 创建角色、外交、锻造、贸易、征服土地,在巨大沙盒里拉起军队并指挥战斗。[Steam 页面](https://store.steampowered.com/app/261550/Mount__Blade_II_Bannerlord/) 同时强调“raise armies”“manage your fiefdom”“establish your clan”等要素。

可提炼为 5 个驱动力:

| 驱动力 | 玩家为什么继续玩 | 可迁移到灵枢 |
|---|---|---|
| 从无名小卒到领主 | 身份阶层不断提升 | 从散修 -> 队长 -> 掌门 -> 一方道主 |
| 每场战斗有战略意义 | 打赢会得俘虏、钱、声望、地盘 | 打赢夺灵脉、弟子、材料、路线安全 |
| 世界持续运转 | 势力互打,玩家可插手 | 五大道统、妖潮、魔修会争夺据点 |
| 队伍成长 | 招兵、升级、换兵种 | 招弟子、养道兵、升阵型 |
| 地盘经营 | 城镇、村庄、贸易、税收 | 山门、灵田、矿脉、丹坊、藏经阁 |

它的关键不是“战斗好玩”这么简单,而是:

```text
战斗是改变世界状态的工具。
```

### 1.2 修仙游戏给出的长期目标

《了不起的修仙模拟器》强调重建宗门、培养弟子、研究法术、收集神器、面对上古危险与其他门派;其 Steam 页面列出宗门建设、学生管理、扩张领域、贸易互动、研究典籍、猎杀妖兽等要素。[Steam 页面](https://store.steampowered.com/app/955900/Amazing_Cultivation_Simulator/)

《鬼谷八荒》强调开放世界、修仙突破、山海经妖兽、命运选择、独立 NPC 世界和玩家干预。[Steam 页面](https://store.steampowered.com/app/1468810/_Tale_of_Immortal/)

修仙题材真正让玩家持续玩的不是“打怪”,而是:

| 目标 | 情绪 |
|---|---|
| 破境 | 我变成了更高层次的存在 |
| 拿机缘 | 我撞到了别人没有的运气 |
| 夺秘境 | 我拿到了稀缺资源 |
| 建宗门 | 这片世界有我的根 |
| 收徒弟 | 我的选择会传承 |
| 抗天劫 | 我在赌命 |

### 1.3 简单上瘾游戏的共同点

几个看起来简单但留存强的游戏有共同结构:

| 游戏 | 核心做法 | 对灵枢的启发 |
|---|---|---|
| Vampire Survivors | 极简操作,一局内快速滚雪球,死亡后带金币升级;官方介绍为极简玩法 + roguelite 元素。[Steam 页面](https://store.steampowered.com/app/1794680/Vampire_Survivors/) | 小局内必须快速变强,不要让玩家等长文本 |
| Balatro | 简单扑克规则 + Joker 组合,一局内构筑爆炸;官方强调强协同和夸张 combo。[Steam 页面](https://store.steampowered.com/app/2379780/Balatro/) | 技能/法宝/弟子要形成“看得懂的组合爆炸” |
| Slay the Spire | 路线选择、卡牌构筑、遗物互动;官方强调每次路线变化和遗物强互动。[Steam 页面](https://store.steampowered.com/app/646570/Slay_the_Spire/) | 地图节点选择必须有风险/收益 |
| Melvor Idle | 把 RuneScape 式冒险抽象成技能互相服务的 idle RPG;官方强调 20+ 技能互相作用和离线成长。[Steam 页面](https://store.steampowered.com/app/1267910/Melvor_Idle/) | 长线系统要简单、可挂机、互相喂养 |
| Luck be a Landlord | slot + roguelike deckbuilder,收集符号形成协同;Nintendo 页面强调随机刺激 + 策略深度。[Nintendo 页面](https://www.nintendo.com/us/store/products/luck-be-a-landlord-switch/) | 可把“机缘”做成简单抽取与组合 |
| Backpack Battles | 背包摆放 + 自动战斗,Steam 标签集中在库存管理、auto battler、异步 PvP。[Steam 页面](https://store.steampowered.com/app/2427700/_Backpack_Battles/) | 法宝/阵法/弟子站位可以成为轻量构筑 |

它们共同点:

```text
规则入口极简单。
选择结果很快反馈。
每局都能形成一点不同的构筑。
长期解锁让玩家想再试一次。
```

## 2. 当前问题诊断

### 2.1 用户没有打怪欲望

原因不是怪物不够多,而是怪物没有战略意义。

当前怪物更像:

```text
地图上的经验包 + 掉落包
```

玩家会问:

```text
为什么我要打这只?
打完除了数字增加,世界有什么变化?
我不打会怎样?
```

新方案必须让每个战斗目标至少绑定一种战略意义:

| 战斗目标 | 战斗意义 |
|---|---|
| 妖巢 | 清掉后附近灵田恢复产出 |
| 魔寨 | 打掉后商路安全,税收提高 |
| 灵矿守卫 | 打赢获得矿脉控制权 |
| 叛修小队 | 打赢可俘虏/招降弟子 |
| 宗门巡逻队 | 打赢改变势力关系和边境归属 |
| 秘境守关 | 打赢进入下一层并拿局内祝福 |

### 2.2 社交信息冗余

当前真实 AI 公司化身、Boss 名、人名、宗派梗太多。AI 圈内玩家可能会懂,普通玩家记不住。

建议:

```text
把“人名社交”降级,把“势力颜色、地盘、图标、功能”升级。
```

玩家不需要记住 21 个名字,只需要先记住:

| 势力 | 玩家记忆点 |
|---|---|
| 沧澜剑派 | 剑、暴击、戒律、北境 |
| 天机阁 | 机关、稳定、商贸、中原 |
| 玄机宗 | 推演、低成本、边境 |
| 青冥派 | 防御、典籍、山岳 |
| 月隐宫 | 速度、暗杀、夜路 |
| 妖潮 | 会吞地盘的野外威胁 |
| 魔修 | 会劫商路和弟子的敌对人类 |

NPC 人名只保留少数:

- 每个势力 1 个掌门。
- 每个玩家队伍 3-6 个核心弟子。
- Boss 只在对应大事件出现。

### 2.3 新鲜感一过没有耐心

因为当前长期目标不够外显。用户需要看到一个远目标:

```text
我要从散修小队,变成拥有 3 座灵脉、12 名弟子、1 座山门的宗门。
```

而不是:

```text
我要把等级从 23 练到 24。
```

### 2.4 数值严重不合理

当前局部数值已经出现:

- HP/灵气进入几千、几万。
- 战斗中等级和属性跳跃明显。
- 灵气上限过大,卡牌消耗失去约束。
- 伤害和敌人血量比例不稳定。

问题本质:

```text
数值膨胀过早,且缺少清晰的战力分层。
```

新方案要把数值拆成两层:

| 层级 | 用途 | 数值范围 |
|---|---|---|
| 个人战斗数值 | 单场战斗读条、伤害、技能 | 50-999 |
| 势力/队伍战力 | 队伍规模、据点战、地图压制 | 100-99999 |

玩家界面优先看个人数值和队伍人数,大数只用于势力面板。

## 3. 新核心玩法: 修仙版骑砍

### 3.1 玩家身份

玩家不再只是“执笔者单人修士”,而是:

```text
灵脉断绝后的无名执笔者,带着自己的 API Key 接入天外灵脉。
起初只有自己一人,后来招募弟子、收编道兵、建立山门,最后参与五大道统和妖魔潮的大陆战争。
```

这个身份天然支持:

- 单人成长。
- 队伍成长。
- 地盘成长。
- 宗门成长。
- 本命书记录大事件。

### 3.2 世界地图

把现在的怪物散点地图改成区域地图:

```text
灵枢大陆
  - 5 个道统主城
  - 12 个族群巢穴区
  - 30-60 个可争夺灵脉/资源点
  - 若干商路
  - 若干秘境入口
```

地图目标不再叫“怪物”,而叫“事件点”:

| 地图点 | 类型 | 玩家动机 |
|---|---|---|
| 狐妖林 | 妖巢 | 清剿后获得皮毛、机缘、区域安全 |
| 黑水魔寨 | 魔修据点 | 获得俘虏、功法、声望 |
| 青玉灵矿 | 资源点 | 占领后每日产矿 |
| 断剑商路 | 护送/劫掠 | 赚灵石、提高贸易 |
| 遗落洞府 | 秘境 | 一局构筑玩法 |
| 沧澜边哨 | 势力据点 | 加入战争、夺边境 |

### 3.3 每场战斗的地图后果

| 战斗结果 | 后果 |
|---|---|
| 胜利 | 资源点归属变化,获得战利品,队伍声望提升 |
| 惨胜 | 得奖励但弟子受伤,需要休整 |
| 失败 | 被俘/丢物资/队伍士气降低,但可触发败笔章 |
| 放过敌人 | 获得人情或情报 |
| 招降敌人 | 获得弟子/道兵 |
| 清剿妖巢 | 降低区域妖潮压力 |

这样玩家打怪有明确欲望:

```text
我要打这窝妖,因为它堵着商路。
我要打这个魔寨,因为里面有我缺的炼器材料。
我要抢这条灵脉,因为它能供养更多弟子。
```

## 4. 大背景重构

### 4.1 世界观主线

原设定“门派 = LLM 厂商”保留,但降噪。不要让玩家记公司梗,要让玩家记势力功能。

新背景:

```text
灵枢大陆原本由五大道统共同维持天书秩序。
某日天外灵脉断裂,妖潮与魔修趁机扩张。
五大道统互不信任,各守一方。
玩家作为执笔者,能用自己的 API Key 重新接入天外灵脉。
谁控制灵脉,谁就能书写世界。
```

核心冲突:

| 冲突 | 游戏作用 |
|---|---|
| 道统争灵脉 | 势力战争 |
| 妖潮吞地盘 | 野外压力和清剿目标 |
| 魔修劫商路 | 护送/讨伐/俘虏 |
| 秘境出世 | 一局 Roguelite 玩法 |
| 天书失衡 | 主线大事件 |

### 4.2 只保留少量可记住名字

命名原则:

```text
玩家先记势力,再记据点,最后才记人。
```

建议保留:

- 五大道统名称。
- 12 个族群名称。
- 每个道统 1 个掌门。
- 每个族群 1 个王。
- 玩家自己的弟子,用标签强化记忆。

弟子不要先给复杂名字,先给功能标签:

| 显示 | 记忆方式 |
|---|---|
| 阿青 · 剑徒 | 暴击输出 |
| 小炉 · 丹童 | 炼丹辅助 |
| 墨七 · 斥候 | 探路和闪避 |
| 铁山 · 护法 | 前排防御 |

## 5. 新循环设计

### 5.1 30 秒循环: 路上有事

```text
选择目标 -> 自动行进 -> 遭遇事件 -> 选择处理 -> 获得小奖励/风险
```

事件示例:

- 遇到受伤散修,救或抢。
- 商队被妖群围住,护送或旁观。
- 发现灵草,采集或派弟子驻守。
- 魔修挑衅,战斗或交涉。

### 5.2 3 分钟循环: 一次出征

```text
接悬赏/选目标
  -> 带队出征
  -> 1-3 个路上事件
  -> 目标战斗
  -> 战利品 + 地图状态变化
  -> 回山门整备
```

这是主循环。玩家每天上来能完成 1-3 次。

### 5.3 10 分钟循环: 一次秘境/攻寨

```text
进入秘境或攻打据点
  -> 5-8 个节点
  -> 选择祝福/战利品/弟子站位
  -> 小 Boss
  -> 结算大章
```

这里借鉴 Balatro / Slay the Spire:

- 每个节点都给选择。
- 每次选择影响本局构筑。
- 每次失败都学到东西。

### 5.4 长期循环: 建宗门

```text
打资源点 -> 养弟子 -> 升建筑 -> 解锁兵种/功法 -> 打更强据点 -> 扩地盘
```

长期目标要一眼可见:

| 阶段 | 目标 |
|---|---|
| 散修 | 活下来,招第一个弟子 |
| 小队 | 5 人队伍,清第一个妖巢 |
| 山门 | 建立宗门,拥有 1 条灵脉 |
| 宗主 | 拥有 3 个据点,参与道统战争 |
| 道主 | 统一一州,抗衡妖潮 |
| 天书主 | 改写灵枢大陆秩序 |

## 6. 战斗玩法重构

### 6.1 从单人卡牌战改为小队战

战斗仍可保留卡牌和 LLM,但加入队伍层:

```text
主角招式 + 弟子技能 + 道兵阵型
```

战斗界面简化为:

| 区域 | 内容 |
|---|---|
| 左侧 | 我方 1 主角 + 3 弟子/道兵位 |
| 右侧 | 敌方队伍/据点 |
| 底部 | 3-5 张主角决策卡 |
| 顶部 | 士气、战损、斩杀线 |

### 6.2 弟子定位

弟子不是复杂 NPC 社交,而是可培养单位。

| 类型 | 作用 | 记忆点 |
|---|---|---|
| 剑徒 | 单点输出 | 秒杀精英 |
| 护法 | 前排承伤 | 保队伍 |
| 丹童 | 回复/炼丹 | 降低出征成本 |
| 斥候 | 探路/先手 | 避免伏击 |
| 符师 | 控制/破防 | 打据点 |
| 器师 | 装备/阵法 | 队伍构筑 |

弟子有名字,但 UI 先显示类型和星级:

```text
阿青  剑徒  ★★
小炉  丹童  ★
铁山  护法  ★★★
```

### 6.3 道兵系统

借鉴骑砍“招兵、升级兵种”,但简化。

| 道兵 | 来源 | 克制 |
|---|---|---|
| 村勇 | 村镇招募 | 低成本 |
| 剑卒 | 沧澜影响区 | 高暴击 |
| 机弩手 | 天机影响区 | 远程稳定 |
| 丹修 | 青冥/山门 | 回复 |
| 夜行者 | 月隐影响区 | 先手 |
| 降魔卫 | Boss 线 | 抗妖魔 |

道兵只显示数量和阶级,避免复杂微操:

```text
剑卒 12/12
机弩手 8/10
丹修 3/3
```

### 6.4 战斗动机

每场战斗前必须显示:

```text
打赢你会获得什么?
打输你会失去什么?
不打会发生什么?
```

示例:

```text
黑水魔寨
胜利: 获得魔血 x3, 俘虏 1-2, 商路安全 +12%
失败: 损失 20% 物资, 一名弟子重伤
不处理: 2 天后附近灵矿产出 -30%
```

这比“Lv.30 毒蟒王”更能刺激行动。

## 7. 数值重做

### 7.1 总原则

```text
低数值、强分层、少属性、可预测。
```

不要让普通玩家看到 1.4 万灵气、2700 HP、1700 攻这种早期数字。大数字留给势力战力。

### 7.2 角色个人数值

保留 4 个主属性:

| 属性 | 作用 |
|---|---|
| 攻 | 直接伤害 |
| 御 | 减伤与护盾 |
| 法 | 灵力、技能效果、成章质量 |
| 速 | 先手、闪避、行军 |

衍生:

| 衍生 | 范围 |
|---|---|
| HP | 80-999 |
| 灵力 | 30-200 |
| 暴击 | 5%-45% |
| 士气 | 0-100 |

### 7.3 等级和境界

等级不再直接暴涨战力。境界负责解锁上限和玩法。

| 境界 | 等级 | HP 参考 | 攻参考 | 解锁 |
|---|---:|---:|---:|---|
| 炼气 | 1-10 | 80-140 | 10-25 | 小队、悬赏 |
| 筑基 | 11-25 | 140-240 | 25-45 | 弟子、资源点 |
| 金丹 | 26-45 | 240-420 | 45-75 | 山门、道兵 |
| 元婴 | 46-70 | 420-650 | 75-120 | 据点战 |
| 化神 | 71-100 | 650-999 | 120-180 | 州域战争 |

不要急着做到飞升。长期阶段可以未来扩展。

### 7.4 经验曲线

建议:

```text
升级所需经验 = round(80 * level^1.35)
```

境界突破需要额外条件:

| 条件 | 目的 |
|---|---|
| 等级达标 | 基础成长 |
| 材料/丹药 | 打怪和炼制动机 |
| 灵脉控制 | 地图目标 |
| 破境战 | 仪式感 |

### 7.5 伤害公式

```text
基础伤害 = 攻 * 技能倍率
减伤 = 100 / (100 + 御 * 3)
最终伤害 = 基础伤害 * 减伤 * 随机浮动(0.9-1.1) * 暴击
```

示例:

```text
攻 30, 技能 1.2x, 敌御 12
基础 36
减伤 100/(100+36)=0.735
最终约 26
```

早期敌人 HP 80-150,玩家 3-5 回合击杀,节奏合理。

### 7.6 队伍战力

队伍战力用于地图判断,不要和单人伤害混用。

```text
队伍战力 =
主角战力
+ 弟子战力总和
+ 道兵数量 * 阶级系数
+ 士气修正
+ 装备/阵法修正
```

示例:

```text
主角 120
弟子 80 + 60
剑卒 10 人 * 12 = 120
士气 90 -> *1.08
总战力约 410
```

地图点直接显示:

```text
敌方战力 360, 胜率较高
敌方战力 590, 胜率危险
```

### 7.7 经济数值

三种资源足够:

| 资源 | 用途 |
|---|---|
| 灵石 | 招募、维护、交易 |
| 材料 | 炼丹、炼器、建筑 |
| 灵脉值 | 成章、突破、山门供养 |

不要再引入太多币种。Token 消耗仍叫燃灵墨,但它是高级行为成本,不是普通战斗货币。

## 8. 让用户想打的目标系统

### 8.1 悬赏板

主城第一屏放悬赏板,替代一堆入口。

悬赏格式:

```text
[清剿] 狐妖林骚扰商路
难度: ★★
奖励: 灵石 80, 白兽皮 5, 区域安全 +10
额外: 首次完成解锁狐妖林巡逻路线
```

每天刷新 5 个,玩家只需要看懂:

- 难度
- 奖励
- 地图影响

### 8.2 地图威胁倒计时

让玩家“不打会有后果”。

示例:

```text
黑水魔寨 2 天后劫掠青玉商路
狐妖林妖潮 3 天后扩散到灵田
天机阁与沧澜边境 1 天后开战
```

这给用户打怪理由:

```text
我不是为了刷经验,我是为了阻止地图变坏。
```

### 8.3 资源点占领

资源点是最强打怪动机。

| 资源点 | 产出 |
|---|---|
| 灵田 | 草药 |
| 灵矿 | 矿石/炼器 |
| 古战场 | 道兵经验 |
| 藏经残阁 | 功法碎片 |
| 商路驿站 | 灵石 |

占领后每天结算,被妖魔或敌对势力攻击。

### 8.4 弟子招降

魔修、散修、敌对小队都可出现可招降对象。

玩家会为了“好苗子”主动战斗:

```text
此队伍中有一名二星丹童,击败后 35% 可招降。
```

这比纯掉落更有吸引力。

## 9. 社交系统减法

### 9.1 删除复杂人名依赖

不再要求玩家记住大量 AI 公司化名。人名只在本命书里作为 flavor,核心 UI 不依赖人名。

错误方向:

```text
墨魔·林泽 与 拥抱真人、玄道魔主、转折天尊存在羁绊
```

新方向:

```text
沧澜叛徒线 2/5
奖励: 解锁暴击流派法宝
下一目标: 击破黑墨剑冢
```

### 9.2 势力关系只保留 3 个状态

| 状态 | 含义 |
|---|---|
| 友好 | 可贸易、可招兵、可学技能 |
| 中立 | 正常通行 |
| 敌对 | 可能被拦截、可攻打 |

具体数值放详情页,主界面不展示 76/100 这类冗余数字。

### 9.3 弟子关系只保留“忠诚/伤势/特长”

不要做复杂好感网。弟子只关心:

| 字段 | 用途 |
|---|---|
| 忠诚 | 是否叛逃 |
| 伤势 | 是否能出征 |
| 特长 | 战斗/炼丹/探路 |

## 10. 简单上瘾结构: 灵脉远征

### 10.1 一局远征

参考 Slay the Spire / Balatro / Vampire Survivors,做一个轻量一局玩法:

```text
选择远征目标
  -> 5 个节点
  -> 每节点 1 个选择
  -> 获得局内祝福
  -> 打据点首领
  -> 战利品带回山门
```

每局 8-12 分钟。

### 10.2 局内祝福

| 祝福 | 效果 |
|---|---|
| 剑阵初成 | 剑徒伤害 +20% |
| 丹火不灭 | 每战后回复 15% HP |
| 机弩连发 | 机弩手首次攻击两次 |
| 夜行无声 | 路上伏击概率 -30% |
| 妖血沸腾 | 战损提高,掉落 +1 |

### 10.3 局内商店

玩家可以用临时资源买:

- 道兵补员。
- 丹药。
- 一次性符纸。
- 情报:看见后面节点。

### 10.4 局外成长

远征结束带回:

- 灵石。
- 材料。
- 弟子经验。
- 据点影响。
- 本命书大事件。

这样玩家会有:

```text
再打一局,也许这次祝福组合能打过那个据点。
```

## 11. LLM 使用策略重构

### 11.1 不要让 LLM 承担高频爽感

高频战斗爽感应靠:

- 数字。
- 阵容。
- 战利品。
- 地图变化。
- 构筑组合。

LLM 只负责:

| 场景 | 作用 |
|---|---|
| 拜入/建宗 | 仪式感 |
| 首次攻下据点 | 记忆点 |
| 弟子入门/叛逃 | 情感点 |
| 境界突破 | 高光 |
| 战役总结 | 玩家本命书 |

### 11.2 分层消耗

| 行为 | 是否消耗 API | 说明 |
|---|---|---|
| 普通战斗数值结算 | 不额外调用 LLM | 用户已验证 Key,但不强制每场调用 |
| 首遇/精英/Boss | 调用短叙事 | 价值更高 |
| 攻城/建宗/破境 | 调用长叙事 | 高仪式 |
| 手动成章 | 玩家确认后调用 | 成本透明 |

仍然不做免 Key 试玩,只是把 Key 消耗集中到高价值时刻。

### 11.3 本命书改成“战役史”

本命书不要记录每只小怪,而记录:

- 第一次招到弟子。
- 第一次夺灵脉。
- 第一次惨败。
- 第一次建宗。
- 第一次灭魔寨。
- 第一次破境。

玩家会更愿意回看。

## 12. MVP 重构路线

### Phase 1: 地图目标替换

目标: 让玩家知道为什么打。

内容:

- 把探索地图上的怪物点包装成“妖巢/魔寨/资源点/商路事件”。
- 每个点显示胜利奖励、失败代价、不处理后果。
- 加悬赏板。
- 去掉主城社交噪音,第一屏只放悬赏、队伍、山门资源。

验收:

- 玩家不用读文档也能回答“我为什么要打这个点”。

### Phase 2: 小队与弟子

目标: 从单人刷怪变成带队成长。

内容:

- 新增 3 类基础弟子: 剑徒、护法、丹童。
- 新增道兵数量。
- 战斗结算影响弟子经验和伤势。
- 可招降散修/魔修。

验收:

- 玩家愿意为了招一个好弟子去打目标。

### Phase 3: 资源点和山门

目标: 战斗改变长期资源。

内容:

- 山门建筑: 丹房、炼器坊、藏经阁、演武场。
- 资源点每日产出。
- 妖潮会攻击资源点。
- 玩家需要防守和巡逻。

验收:

- 玩家第二天回来能看到地盘产出和威胁变化。

### Phase 4: 灵脉远征

目标: 做出 8-12 分钟可重复的一局玩法。

内容:

- 5-8 节点远征。
- 局内祝福。
- 局内商店。
- 首领战。
- 战役总结章。

验收:

- 玩家愿意连续开第二局。

### Phase 5: 势力战争

目标: 接近骑砍的世界格局变化。

内容:

- 五大道统争夺边境资源点。
- 玩家可加入、雇佣、背叛、自立。
- 地图显示势力颜色变化。
- 攻下 3 个资源点可建宗。

验收:

- 玩家能讲出“这是我的地盘”。

## 13. UI 重构建议

### 13.1 主城第一屏

只保留 4 块:

```text
队伍状态
悬赏板
山门资源
世界威胁
```

移除第一屏复杂内容:

- 复杂人缘。
- 大量资质说明。
- 长列表入口。
- 过多日志。

### 13.2 地图点卡片

每个目标卡必须统一:

```text
名称: 黑水魔寨
类型: 魔修据点
敌方战力: 380
推荐战力: 320+
胜利: 魔血 x3 / 俘虏 1 / 商路安全 +12
失败: 物资 -20% / 弟子可能受伤
倒计时: 2 天后劫掠商路
```

### 13.3 队伍面板

```text
主角: 执笔者 金丹
弟子: 阿青(剑徒) 小炉(丹童) 铁山(护法)
道兵: 剑卒 12 / 机弩手 6
士气: 82
粮草: 5 天
战力: 430
```

简单、清楚、能决策。

## 14. 这个方案的吸引点

### 14.1 玩家持续玩的理由

| 时间 | 理由 |
|---|---|
| 前 5 分钟 | 我接了悬赏,打掉第一个妖巢 |
| 前 30 分钟 | 我招到第一个弟子,抢到第一条灵脉 |
| 第 1 天 | 我建了小山门,有每日产出 |
| 第 3 天 | 妖潮来打我的灵田,我要防守 |
| 第 7 天 | 我能参与道统战争,抢边境据点 |
| 第 30 天 | 我有自己的宗门、弟子谱和本命书战役史 |

### 14.2 打怪欲望从哪里来

不再是:

```text
这只怪给经验。
```

而是:

```text
这只怪占着我的矿。
这伙魔修里有好弟子。
这个妖巢不清会吞我的灵田。
这个据点打下来我就能建宗。
```

### 14.3 社交怎么简化

不再记 21 个 Boss 名,只记:

- 5 个道统。
- 自己的 3-6 个弟子。
- 地图上正在威胁你的几个据点。

### 14.4 AI 新鲜感过后还剩什么

还剩:

- 地图扩张。
- 队伍构筑。
- 弟子养成。
- 资源点防守。
- 一局远征。
- 宗门成长。

AI 变成高光记录器,不是唯一玩法支柱。

## 15. 最终建议

如果只做小修小补,《灵枢笔录》会继续卡在:

```text
概念很酷,但我为什么还要继续打?
```

建议直接转向:

```text
修仙版骑砍 + 轻量一局构筑 + BYOK 本命书高光记录。
```

最小可行版本只需要做 4 件事:

```text
1. 地图点从“怪物”改成“有后果的目标”。
2. 主角从单人改成“小队/弟子/道兵”。
3. 战斗奖励从经验材料改成“资源点、弟子、地盘、安全度”。
4. 主城从信息展示改成“悬赏板 + 山门经营”。
```

这四件事完成后,用户才会真正产生持续动机:

```text
我不是来刷怪的。
我是来把这片灵脉大陆打成我的宗门版图的。
```
