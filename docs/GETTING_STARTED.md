## 🚀 启动游戏

### 方式 A:一键启动(推荐)

```bash
cd ~/项目/LingshuCodex
./start.sh
```

脚本会自动:
- 创建 Python venv + 装后端依赖
- npm install 前端依赖
- 同时启动后端(:8020) + 前端(:5173)

### 方式 B:手动启动两个窗口

```bash
# Terminal 1 - 后端
cd ~/项目/LingshuCodex/backend
source .venv/bin/activate
uvicorn app.main:app --host 127.0.0.1 --port 8020 --reload

# Terminal 2 - 前端
cd ~/项目/LingshuCodex/frontend
npm run dev
```

### 验证启动成功

打开浏览器访问 **http://127.0.0.1:5173/**

你应该看到:
- 暗色背景 + 金色"灵"字徽章 logo
- 顶部"bobdong.cn 推荐"广告卡
- 一个填 API Key 的表单

## ❓ 常见问题

### Q1:打不开 http://127.0.0.1:5173?
检查:
1. 终端有没有报错(端口被占?)
2. 浏览器代理插件(关掉无痕试试)
3. `lsof -i :5173` 看进程在不在

### Q2:验证模型时全失败?
- 检查 API Key 是否正确(末尾 4 位粘贴对了吗)
- 检查 API 地址(是 `/v1` 后缀吗?)
- 看后端日志 `/tmp/lingshu_backend.log`

### Q3:战斗中 LLM 报错?
游戏会**自动 fallback 到预设叙事模板**,不会中断。看日志有 `[LLM Fallback]` 字样即正常。

### Q4:战斗中我刷新页面了?
战斗状态会丢失,自动跳回主城。**战斗时别刷新**。

### Q5:想用 OpenAI 官方而非 bobdong.cn?
入门页 base_url 输入框点「OpenAI 官方」预设按钮 → `https://api.openai.com/v1`,再填 OpenAI Key。

### Q6:可以多人玩吗?
**MVP 版本不可以**,单玩家 demo_player 硬编码。后续版本会加 JWT 多用户。
