# 灵枢笔录 · 游戏说明

> **一句话**:这是一款 Web 文字 RPG。门派 = LLM 厂商,境界 = 模型档位,战斗叙事由你自己的 API Key 实时生成。

---

## 🎮 这是什么游戏

一款**披着国风修仙皮的 AI 原住民修仙游戏**。

```
传统手游        →   本游戏
─────────────────────────────────
动画战斗         →   AI 实时生成的文学叙事(每场都不重复)
固定卡牌        →   按门派分化的招式 + 笔魂调味
6 个职业        →   5 大门派 × 9 个境界 + LLM 模型档位
平台烧 API 钱   →   玩家自带 Key,Token 即灵气
```

**你不是在"看 AI 生成内容"——你是在用自己的 API Key,通过游戏调用 LLM。**

## 🌟 核心循环

```
① 打开游戏 → 填 API Key(默认推荐 bobdong.cn 网关)
② 系统探测 Key 可调用的模型 → 自动告诉你能选哪派
③ 选门派 → 流式实测该派每个模型 → 全过创角
④ 主城 → 选「开始修行」进入仿真地图
⑤ 12 只怪物在地图上自由游荡,鼠标悬停看故事
⑥ 点击怪物开战 → LLM 实时生成 200 字仙侠叙事
⑦ 胜利掉落物品 → 进背包,丹药可用,法宝待装备
⑧ 突破境界 = 升级 LLM 模型档位(Haiku → Sonnet → Opus)
```

## 🤖 核心技术亮点

### 1. **LLM 是游戏的"心脏",不是"皮肤"**
每场战斗都会真的调用一次 LLM(除缓存命中),你能看到自己的 token 流过去。

### 2. **门派 = 厂商,境界 = 模型**
你选沧澜剑派 = 加入 Anthropic 修真界。突破境界相当于升级模型(Haiku → Opus),叙事质量随之提升。

### 3. **BYOK 经济模型**
平台不烧 token——你用自己的 Key 推动战斗。这意味着:
- 平台可持续(不会因为成本关服)
- 你完全可控自己的开销
- "全网源头 token,更真、更快、更强"(推荐 bobdong.cn 网关)

### 4. **零动画美术**
不是省钱,是设计哲学:
- 美术成本接近零
- 战斗永不重复(LLM 实时叙事)
- 加载快,响应快
- 兼容性极好(纯 Web)

### 5. **真公司化身 Boss + 真实故事**
21 位 Boss 来自 18 家真实大模型公司。
- 阿莫迪克斯(沧澜山主)= Anthropic 现实掌门
- 山姆道君(天机阁掌教)= OpenAI Sam Altman
- 王小川 = 百川海主
- 玄道魔主 = xAI / Grok / 马斯克
- ...
每位 Boss 的故事都基于真实公司背景演绎,带羁绊与冲突。
