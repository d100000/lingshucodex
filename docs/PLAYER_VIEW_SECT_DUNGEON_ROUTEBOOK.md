# 《灵枢笔录》玩家视角门派副本路线书

> 日期: 2026-05-27  
> 目标: 以玩家视角写清楚“我选择不同门派时,会进入哪些副本、每个副本发生什么、为什么要打、打完如何推进四宗统一”。  
> 口径: 每个周目采用“本门 + 三个目标外宗 = 本局四宗”。第五宗作为幕后观察者或隐藏线索,不在首轮三个归盟副本中强行塞给玩家。  
> 关联文档: `SECT_DUNGEON_UNIFICATION_CAMPAIGN_DESIGN.md`、`PLAYER_BACKGROUND_AND_FIXED_BONDS_DESIGN.md`、`SECT_DISCIPLE_CHARACTER_BIBLE.md`。

## 0. 玩家会怎样理解副本

玩家看到的不是“副本 1、副本 2、副本 3”,而是一张清楚的归盟路线:

```text
我是某个门派的弟子。
其它三个门派不信我、不服我、或者欠我一个解释。
我需要进入他们的地界,解决他们的旧案,通过他们的试炼。
每通关一个门派正传副本,我就拿到一枚盟印。
三枚盟印集齐后,我可以开启最终副本: 四宗归盟台。
```

副本的玩家动机:

| 玩家问题 | 副本给出的回答 |
|---|---|
| 我为什么要去这个门派? | 因为它握着我身世、破境或统一四宗的一块关键证据 |
| 我为什么要打这里的怪? | 因为这些怪是机关失控、心魔、旧案怨念、守关试炼的具象化 |
| 我打完有什么变化? | 门派态度改变,关键人物加入熟人池,拿到盟印,下一轮事件池变化 |
| 我为什么继续打下一个副本? | 因为三枚盟印才能开最终归盟台 |

---

## 1. 共用副本规则

每个门派正传副本都用同一套节奏,让玩家不用重新学习系统:

| 节点 | 玩家感受 | 玩法内容 |
|---|---|---|
| 1. 入境 | 我进入目标门派地界 | 短剧情 + 关键人物登场 |
| 2. 起疑 | 我发现这事和我有关 | 线索、抉择或轻战斗 |
| 3. 试炼一 | 我开始被这个门派考验 | 普通战斗或门派机制 |
| 4. 试炼二 | 我拿到更深线索 | 精英战斗或机关/辩经 |
| 5. 问心 | 关键人物质问我 | 选择立场,影响结局 |
| 6. 守关 | 我打副本 Boss | 核心战斗 |
| 7. 归盟 | 我拿到盟印或留下裂痕 | 结算、奖励、世界状态变化 |

每个副本有三类结局:

| 结局 | 含义 |
|---|---|
| 归盟 | 该门派愿意签盟,后续支援玩家 |
| 压服 | 该门派暂时服从,但关系有裂痕 |
| 留证 | 玩家拿到证据和盟印,但门派内部还没统一 |

## 2. 我选择沧澜剑派

我的身份:

```text
我是沧澜掌刑堂收养的白石渡旧案遗孤。
我身上有林泽剑意残痕。
沧澜教我握剑,也在审我。
```

我的统一目标:

```text
我必须让沧澜从复仇里走出来。
我要让天机、青冥、月隐承认旧案真相,最终与沧澜重立四宗盟约。
```

路线:

| 顺序 | 副本 | 推荐阶段 | 盟印 |
|---:|---|---|---|
| 1 | 天机阁《机关廊锁心案》 | 筑基前后 | 天机盟印 |
| 2 | 青冥派《旧藏书楼书火案》 | 金丹前后 | 青冥盟印 |
| 3 | 月隐宫《月下名册缺页》 | 元婴前后 | 月隐盟印 |
| 终局 | 《四宗归盟台》 | 化神前后 | 四宗归盟 |

幕后观察者:

```text
玄机宗周虚空。
他每次都像局外人,但我逐渐发现,他也许是白石渡那夜最早知道真相的人。
```

### 2.1 天机阁副本: 《机关廊锁心案》

我为什么要去:

```text
白石渡那夜,天机阁的机关锁曾自行开启。
沧澜怀疑天机阁提前知道林泽叛逃。
我需要进入机关廊,查到那一夜是谁开了锁。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 唐灵枢 | 一上来就把我登记成试验者,像麻烦制造者,但是真心帮我 |
| 鲁照衡 | 帮我检查本命书书扣,发现它和机关锁同频 |
| 罗开阖 | 看守机关锁,不愿让我带走记录 |
| 谷开明 | 天机公开派,希望我把旧档案公之于众 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 我刚进天机阁,唐灵枢就拉我进机关廊,说“来都来了,顺手试个锁” | 跟她进去 / 要求先见长老 / 警惕她 |
| 起疑 | 机关廊识别到我的本命书书扣,自动亮起“白石渡旧档” | 读取旧档 / 让鲁照衡先检查 / 强行关闭 |
| 试炼一 | 失控铜鸢群围攻,考验玩家基础战斗 | 直面战斗 |
| 试炼二 | 三道机关门: 齿轮、星盘、锁芯。每道门给一个临时祝福 | 选攻击祝福 / 选防御祝福 / 选探查祝福 |
| 问心 | 罗开阖要求我把书扣交给天机阁保管 | 交出 / 拒绝 / 提议共同封存 |
| 守关 | Boss“万象锁阵”启动,模拟白石渡那夜的开锁过程 | Boss 战 |
| 归盟 | 我看见旧档: 那夜开锁者不是天机弟子,而是持有月隐名册残页的人 | 公开记录 / 交给谷开明 / 暂时封存 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 万象锁阵 | 每回合切换机关形态,玩家需根据提示选择攻击、蓄气或防守 |

结局影响:

| 选择 | 结果 |
|---|---|
| 公开记录 | 获得天机盟印,天机公开派支持我,守旧派不满 |
| 交给谷开明 | 获得天机盟印和机关支援,但后续可能出现档案外泄事件 |
| 暂时封存 | 获得盟印,天机整体观望,唐灵枢更信任我 |

首通奖励:

- 天机盟印。
- 机关祝福“锁心识破”: 进入副本时可提前看见一个陷阱节点。
- 唐灵枢加入固定熟人池。
- 下一轮新增“机关旧档流传”事件。

### 2.2 青冥派副本: 《旧藏书楼书火案》

我为什么要去:

```text
青冥旧藏书楼百年前失火,月隐宫背锅。
但我在天机旧档里看到,白石渡那夜也有同样的灰烬。
如果书火案和白石渡有关,我必须找到真证据。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 石听雨 | 她先救人,再问是非,是这条线里最温柔的锚点 |
| 文问松 | 带我读旧卷,告诉我青冥不是不能认错,只是害怕真相 |
| 陈千古 | 坚信月隐无罪,愿意和我一起查 |
| 陆正辞 | 当年审案者,他不坏,但他可能判错了 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 我在雨夜入青冥山,石听雨正给受伤弟子包扎 | 帮忙救治 / 直接问案 / 旁观 |
| 起疑 | 文问松带我看残卷,残卷边缘和我的古碑拓片吻合 | 追问残卷来源 / 暂时不说拓片 / 交出拓片 |
| 试炼一 | 守书灵认为我是外宗盗卷者,发起战斗 | 战斗 |
| 试炼二 | 辩经节点: 青冥弟子质问“没有铁证凭什么翻案” | 讲证据 / 讲情理 / 沉默等陈千古作证 |
| 问心 | 陆正辞承认卷宗有缺页,但要求我不要公开 | 答应保密 / 坚持公开 / 要求看缺页 |
| 守关 | Boss“守卷心魔”出现,它由青冥百年不愿认错的执念形成 | Boss 战 |
| 归盟 | 我拿到书火残页,证明月隐至少不是唯一真凶 | 为月隐作证 / 继续查真凶 / 只取青冥盟印 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 守卷心魔 | 防御高,会用“旧律压身”降低玩家速度,需要破防或持续输出 |

结局影响:

| 选择 | 结果 |
|---|---|
| 为月隐作证 | 青冥盟印到手,月隐关系提前改善,青冥守旧派敌意增加 |
| 继续查真凶 | 石听雨和文问松信任提升,最终归盟台可走礼法路线 |
| 只取盟印 | 主线推进最快,但书火案后续会以更糟形式爆发 |

首通奖励:

- 青冥盟印。
- 经卷祝福“守心”: 副本内首次濒死时保留 1 点生命。
- 石听雨羁绊推进。
- 解锁书火残页收集。

### 2.3 月隐宫副本: 《月下名册缺页》

我为什么要去:

```text
天机旧档指向月隐名册。
青冥书火案又证明月隐可能被冤枉。
可沧澜掌刑堂说月隐救我是假,偷我是真。
我必须亲自去月隐宫。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 沈鸦青 | 她危险、直接,逼我面对真相 |
| 叶照眠 | 他一直暗中护我,但从不说自己付出了什么 |
| 顾长忆 | 他能恢复我的记忆,也承认曾封过我的记忆 |
| 夜阑宫主 | 她知道月隐被冤的证据,但不轻易交出 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 沈鸦青在月下渡口拦住我,问我是不是来审月隐的 | 承认 / 否认 / 反问她当年在哪里 |
| 起疑 | 叶照眠交给我一盏旧灯,灯上刻着我的幼名 | 收下 / 质问 / 拒绝 |
| 试炼一 | 月影迷廊,我需要辨别真实道路和记忆幻影 | 靠直觉 / 靠顾长忆提示 / 靠战斗破影 |
| 试炼二 | 名册碎珠释放出白石渡夜的片段,但片段互相矛盾 | 相信沧澜记忆 / 相信月隐记忆 / 暂存疑点 |
| 问心 | 顾长忆承认曾封印我的一段记忆 | 原谅 / 追责 / 要他立刻解封 |
| 守关 | Boss“无名影册”显现,它会变成我熟人的影子 | Boss 战 |
| 归盟 | 夜阑宫主愿交出月隐盟印,条件是沧澜必须承认当年误判过月隐 | 答应作证 / 要完整名册 / 私下与沈鸦青结盟 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 无名影册 | 会复制玩家熟人影像,每复制一次改变技能倾向 |

结局影响:

| 选择 | 结果 |
|---|---|
| 答应作证 | 月隐盟印到手,沧澜掌刑堂压力升高 |
| 要完整名册 | 获得更多身世线索,顾长忆关系下降 |
| 私下结盟 | 沈鸦青羁绊大幅推进,月隐正式归盟延后但最终可走暗盟路线 |

首通奖励:

- 月隐盟印。
- 记忆碎片一枚。
- 月影祝福“辨伪”: 下一轮更容易触发玩家羁绊私聊。
- 沈鸦青或顾长忆进入固定熟人池。

## 3. 我选择天机阁

我的身份:

```text
我是被机关锁认主的天机学徒。
我不是天机阁的工具,但所有人都觉得我像一把钥匙。
```

我的统一目标:

```text
我要证明机关不是控制四宗的锁,而是重建盟约的钥匙。
```

路线:

| 顺序 | 副本 | 推荐阶段 | 盟印 |
|---:|---|---|---|
| 1 | 沧澜剑派《剑冢林泽案》 | 筑基前后 | 沧澜盟印 |
| 2 | 玄机宗《无字天衍图》 | 金丹前后 | 玄机盟印 |
| 3 | 青冥派《青澜古碑注疏》 | 元婴前后 | 青冥盟印 |
| 终局 | 《四宗归盟台》 | 化神前后 | 四宗归盟 |

幕后观察者:

```text
月隐宫。
许藏月和顾长忆一直认为天机阁不该碰名册机关锁。
```

### 3.1 沧澜剑派副本: 《剑冢林泽案》

我为什么要去:

```text
沧澜怀疑天机阁当年送林泽出城。
如果这个嫌疑不洗清,沧澜永远不会和天机归盟。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 陈墨痕 | 他不信我,但他也想知道林泽真相 |
| 墨听潮 | 他和玩家一样是旧案遗孤,容易被真相伤到 |
| 萧问剑 | 曾放走林泽一次,是关键证人 |
| 叶归剑 | 她救过天机弟子,可作为两派缓冲 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 我带着机关旧档入沧澜,陈墨痕要求我先过问剑台 | 接受切磋 / 请求先验档案 / 让叶归剑作保 |
| 起疑 | 剑冢里一柄旧剑对我的机关锁芯有反应 | 拔剑 / 记录反应 / 请墨听潮辨认 |
| 试炼一 | 剑冢残剑化作林泽旧招攻击我 | 战斗 |
| 试炼二 | 我找到南宫飞匣送林泽出城的机关车记录 | 公开记录 / 先问萧问剑 / 暂藏记录 |
| 问心 | 陈墨痕质问: 天机若无罪,为何藏档十七年 | 承认天机藏过 / 反问沧澜是否也藏过 / 道歉 |
| 守关 | Boss“墨剑旧影”出现,模拟林泽出逃当夜 | Boss 战 |
| 归盟 | 萧问剑承认自己也放过林泽,因为林泽说要查黑卷商人 | 公开萧问剑证词 / 保护萧问剑 / 把责任推给林泽 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 墨剑旧影 | 高暴击,会在低血时释放“叛门一剑” |

结局影响:

| 选择 | 结果 |
|---|---|
| 公开证词 | 获得沧澜盟印,陈墨痕信任上升,萧问剑受门内压力 |
| 保护萧问剑 | 获得盟印,沧澜内部裂痕较小,真相推进较慢 |
| 推给林泽 | 快速拿盟印,但墨听潮关系下降,后续林泽线更痛 |

奖励:

- 沧澜盟印。
- 剑意祝福“旧影破绽”: 对剑修敌人首回合命中提高。
- 陈墨痕或墨听潮加入熟人池。

### 3.2 玄机宗副本: 《无字天衍图》

我为什么要去:

```text
天机和玄机都说自己才懂天衍图。
如果我不能让玄机承认天机并非篡图者,两派就不会归盟。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 韩极简 | 他说话很少,但每句话都像题目 |
| 周虚空 | 他知道玩家命格空白的原因 |
| 明素问 | 能分辨图中记忆是真是假 |
| 冯照矩 | 天机核心弟子,与韩极简相爱 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 韩极简递给我一道无字题,答不出就不能入山 | 用机关解 / 用直觉解 / 请冯照矩提示 |
| 起疑 | 无字天衍图浮现半句: “锁非锁,局非局” | 记录 / 质问周虚空 / 询问明素问 |
| 试炼一 | 幻方阵要求玩家在三回合内破局 | 机关路线 / 暴力路线 / 推演路线 |
| 试炼二 | 韩极简和冯照矩的私通信物被发现 | 帮他们隐瞒 / 劝他们公开 / 交给长老 |
| 问心 | 周虚空问我: 天机真的愿意承认自己算错吗 | 承认 / 辩解 / 反问玄机三日不报 |
| 守关 | Boss“无字天衍影”以玩家选择生成不同形态 | Boss 战 |
| 归盟 | 明素问证明两派各持半真,谁独占天衍图都会出错 | 共管天衍图 / 让天机保管 / 让玄机保管 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 无字天衍影 | 会根据玩家前面选择获得机关、推演或情感反制 |

结局影响:

| 选择 | 结果 |
|---|---|
| 共管天衍图 | 获得玄机盟印,韩极简与冯照矩线继续 |
| 天机保管 | 获得盟印但玄机不满,最终归盟偏机关盟约 |
| 玄机保管 | 冯照矩压力上升,但周虚空信任提升 |

奖励:

- 玄机盟印。
- 推演祝福“半步破局”: 副本中一次选择可预览后果。
- 韩极简和冯照矩羁绊线开启。

### 3.3 青冥派副本: 《青澜古碑注疏》

我为什么要去:

```text
青冥认为天机阁改过青澜古碑注疏。
如果我不能修复注疏,青冥会把天机视作篡经者。
```

关键人物:

| 角色 | 我看到的作用 |

| 文问松 | 愿意听我解释机关与经义的关系 |
| 宋扶碑 | 能读古碑第三层刻痕 |
| 范经纬 | 用经纬法校对天机算法 |

副本流程:

### 6.1 沧澜剑派副本: 《掌刑堂赎罪线》

我为什么要去:

```text
沧澜掌刑堂当年误判月隐,但从不公开认错。
如果沧澜不承认旧错,月隐永远不会真心归盟。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 陈墨痕 | 对月隐戒备,但他不是不讲理 |
| 顾千澜 | 夹在沧澜责任和沈鸦青之间 |
| 闻人守墨 | 当年误判月隐的掌刑者 |
| 沈鸦青 | 她想要道歉,但不愿低头求 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 我以月隐弟子身份入沧澜,掌刑堂要求搜身 | 接受 / 拒绝 / 让顾千澜作保 |
| 起疑 | 闻人守墨的旧判卷出现缺页,缺页与名册残页吻合 | 公开缺页 / 私下问他 / 交给陈墨痕 |
| 试炼一 | 掌刑剑阵启动,测试我是否携带妖影 | 战斗 |
| 试炼二 | 顾千澜问我: 沈鸦青是否也想让沧澜难堪 | 替她解释 / 不替她说话 / 反问他的心意 |
| 问心 | 闻人守墨承认自己可能判错,但怕公开后沧澜威信崩塌 | 要求公开 / 给他台阶 / 用问剑压服 |
| 守关 | Boss“掌刑旧剑”代表沧澜不愿认错的剑律 | Boss 战 |
| 归盟 | 沧澜愿给盟印,但要求月隐也交出封忆记录 | 交换记录 / 先拿道歉 / 拒绝交换 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 掌刑旧剑 | 会施加“审判”状态,玩家若连续进攻会被反击 |

结局影响:

| 选择 | 结果 |
|---|---|
| 交换记录 | 获得沧澜盟印,双方关系稳步改善 |
| 先拿道歉 | 沈鸦青信任提高,沧澜保守派敌意上升 |
| 拒绝交换 | 以压服方式拿盟印,最终归盟台冲突更大 |

奖励:

- 沧澜盟印。
- 掌刑卷残页。
- 顾千澜与沈鸦青三角线推进。

### 6.2 青冥派副本: 《书火真凶线》

我为什么要去:

```text
青冥藏书楼的火烧了百年,月隐背了百年。
我不是来求青冥原谅,我是来找真凶。
```

关键人物:

| 角色 | 我看到的作用 |
|---|---|
| 石听雨 | 她害怕真相伤人,但仍会救我 |
| 文问松 | 青冥温和派,愿意听证 |
| 陆回雪 | 月隐证人,需要保护 |
| 陈照古 | 青冥长老,愿意翻案但要铁证 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 青冥弟子不准月隐入藏书楼,石听雨悄悄给我送药 | 收药 / 拒绝 / 请她作证 |
| 起疑 | 文问松发现青冥内卷宗被人改过页码 | 追查改页者 / 保护文问松 / 通知月隐 |
| 试炼一 | 藏书楼守灵把我当纵火者 | 战斗 |
| 试炼二 | 陆回雪的证词被守旧派攻击为伪证 | 让她当众作证 / 暂时保护她 / 用名册佐证 |
| 问心 | 陈照古问我: 月隐若也隐瞒过关键证据,你是否同样公开 | 公开 / 分阶段公开 / 先救人 |
| 守关 | Boss“焚卷真影”现身,它不是月隐术,而是青冥内鬼的残念 | Boss 战 |
| 归盟 | 青冥愿撤销旧判,但要求月隐交出名册对应页 | 交出副本 / 共同保管 / 拒绝交出 |

Boss 设计:

| Boss | 战斗特点 |
|---|---|
| 焚卷真影 | 会燃烧玩家灵力,适合考验蓝条管理 |

结局影响:

| 选择 | 结果 |
|---|---|
| 交出副本 | 获得青冥盟印,青冥公开道歉较快 |
| 共同保管 | 双方关系最稳,最终可走礼法或暗盟路线 |
| 拒绝交出 | 月隐内部满意,青冥只被压服不是真归盟 |

奖励:

- 青冥盟印。
- 真凶线索。
- 石听雨信任根据玩家保护平民程度变化。

### 6.3 天机阁副本: 《名册机关锁》

我为什么要去:

```text
月隐名册最关键的机关锁在天机阁手里。
如果锁不拿回或重修,月隐永远觉得自己被控制。
```

关键人物:

| 沈机月 | 同时被月隐和天机尊为旧师 |
| 顾长忆 | 需要锁来恢复玩家更深记忆 |

副本流程:

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 入境 | 天机阁允许我入锁库,但要求全程被机关眼监控 | 接受 / 摧毁机关眼 / 请求唐灵枢陪同 |
| 起疑 | 名册锁对我产生反应,显示“封忆者并非一人” | 追查封忆者 / 先修锁 / 告诉顾长忆 |
| 试炼一 | 锁库机关影卫启动 | 战斗 |
| 试炼二 | 唐灵枢发现锁被人加过第三重权限 | 查权限 / 保护她 / 让沈机月解释 |
| 问心 | 罗开阖问我: 如果名册能改写人心,月隐凭什么独自保管 | 交回月隐 / 共管 / 由玩家保管 |

| 归盟 | 沈机月提出重修锁: 四宗各持一钥,玩家持总印 | 接受 / 要月隐主钥 / 拒绝天机参与 |

Boss 设计:

| Boss | 战斗特点 |

奖励:

## 7. 最终副本: 《四宗归盟台》

开启条件:

```text
我完成三个外宗正传副本。
我拥有三枚盟印。
至少两名关键人物愿意为我作证。
```

玩家进入时看到:

```text
青澜古碑下方出现第四层碑文。
碑文不是功法,而是一份旧盟约。
四宗祖师曾约定: 天外裂简再临时,不得互相夺简,必须共立执笔者。
而我,就是这一代被推到碑前的人。
```

### 7.1 最终副本流程

| 节点 | 内容 | 玩家选择 |
|---|---|---|
| 三印入碑 | 我将三枚盟印嵌入古碑,前三个副本的选择依次回响 | 无 |
| 旧怨回响 | 每个目标门派派代表质问我曾经的选择 | 解释 / 承认 / 反问 |
| 关键人物作证 | 熟人池里支持度最高的 2-4 人出现 | 看此前羁绊 |
| 幕后门派现身 | 本局未归盟的第五宗揭示一条隐藏情报 | 相信 / 质疑 / 暂存 |
| 四宗问心 | 四宗共同问我: 统一之后谁来保管裂简 | 本门 / 四宗共管 / 玩家执笔 / 暗盟保管 |
| 终战 | Boss“裂简伪盟”出现,它吸收四宗猜疑成形 | Boss 战 |
| 归盟结局 | 根据此前选择生成统一方式 | 结算 |

### 7.2 最终 Boss: 裂简伪盟

战斗意义:

```text
它不是怪物,而是四宗互相不信任的集合。
玩家不是打死某个门派,而是打碎“旧盟被篡改后的假天道”。
```

Boss 阶段:

| 阶段 | 机制 |
|---|---|
| 第一阶段: 旧怨 | 使用玩家前三个副本中关系最低门派的招式 |
| 第二阶段: 伪证 | 随机封锁一个关键人物作证效果 |
| 第三阶段: 裂简 | 根据玩家统一方式召唤不同幻象 |
| 最终阶段: 执笔 | 玩家选择最终盟约形式,获得对应终结技 |

### 7.3 统一结局

| 结局 | 玩家感觉 | 常见条件 |
|---|---|---|
| 四宗重盟 | 我用真相和信任重新立盟 | 多数副本选择公开真相,关键人物支持高 |
| 执笔盟主 | 我用实力压住争端 | Boss 战表现强,但关系线一般 |
| 暗盟共治 | 我保留秘密,让四宗在暗线中合作 | 月隐/玄机相关保密选择多 |
| 礼法盟约 | 我让证据、卷宗和誓约约束四宗 | 青冥线完成度高 |
| 机关盟约 | 我让制度和锁钥防止背叛 | 天机线完成度高 |
| 剑誓盟约 | 我以问剑结果赢得尊重 | 沧澜线战斗和问剑选择突出 |
| 裂盟再起 | 我暂时失败,四宗未能统一 | 盟印齐但关键人物支持不足 |

裂盟不是结束。  
失败后玩家进入“裂盟状态”,可以补做回响副本、修复关系、再次挑战归盟台。

## 8. 玩家视角的主城显示

玩家不该看到策划表,应该看到自己的路线:

```text
我的门派: 沧澜剑派
我的目标: 统一四宗

天机阁: 机关廊锁心案
状态: 已归盟
关键人物: 唐灵枢
盟印: 已获得

青冥派: 旧藏书楼书火案
状态: 调查中
关键人物: 石听雨、文问松
归盟条件: 书火残页 2/3,击败守卷心魔

月隐宫: 月下名册缺页
状态: 敌意
关键人物: 沈鸦青、顾长忆
归盟条件: 未接触
```

推荐按钮文案:

```text
继续青冥线
查看归盟条件
挑战守卷心魔
前往四宗归盟台
```

玩家每次进入副本前都应该知道:

- 这是哪个门派的故事线。
- 我为什么要去。
- 这次通关能不能拿盟印。
- 牵涉哪 2-3 个关键人物。

## 9. 最小可实现路线

第一版不需要一次做完所有细节。  
建议先实现:

| 优先级 | 内容 |
|---|---|
| P0 | 每个门派路线生成 3 个目标副本 |
| P0 | 每个副本 7 个固定节点 |
| P0 | 首通副本发盟印 |
| P1 | 每个副本至少 1 个关键选择改变结局 |
| P1 | 每个副本至少 1 个 Boss 战 |
| P1 | 主城显示四宗归盟进度 |
| P2 | 最终副本根据前三个副本选择生成结局 |

最小副本节点模板:

```text
入境剧情
线索选择
普通战斗
精英战斗
问心选择
Boss 战
盟印结算
```

这样玩家不会觉得自己在刷怪,而是在推进一条清楚的门派统一路线。
