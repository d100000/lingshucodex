# 《灵枢笔录》玩家身世、门派背景与固定羁绊故事线设计

> 日期: 2026-05-27  
> 目标: 让玩家选择不同门派时拥有逻辑自洽的角色背景,并让“下一轮”偶尔触发玩家与固定角色之间的连续羁绊事件。  
> 设计口径: 不是随机人名堆叠,而是以玩家为中心建立可长期推进的友情、爱情、三角关系、师徒、宿敌、旧案和门派纠葛。  
> 关联文档: `SECT_DUNGEON_UNIFICATION_CAMPAIGN_DESIGN.md`、`SECT_DISCIPLE_CHARACTER_BIBLE.md`、`GAMEPLAY_REDESIGN_BANNERLORD_CULTIVATION.md`、`CULTIVATION_LEVEL_PROGRESSION_REDESIGN.md`。

---

## 0. 核心设计结论

玩家不能只是“选一个门派,然后开始刷怪”。  
修仙 RPG 的长期吸引力来自:

```text
我是谁?
谁在保护我?
谁在利用我?
谁因为我改变命运?
我选了这个门派,会让哪些旧关系变好或变坏?
```

因此主角背景应采用“同一核心身世 + 五种门派解释”的结构。

统一核心:

```text
玩家是白石渡裂简夜的幸存者。
那一夜,天外裂简坠入青澜古碑,五宗同时赶到。
玩家本命书被烧成空白,命格无法被完整推演。
五宗都认为玩家身上藏着通向天外灵脉的钥匙,但各宗解释不同。
```

选择门派时,不是改写身世,而是决定:

```text
谁先收留玩家;
谁掌握玩家旧案的第一份证据;
谁会成为玩家的师门、保护者和债主;
谁会因此怀疑玩家、接近玩家、爱上玩家或背叛玩家。
```

## 1. 世界背景补全

### 1.1 天外裂简

天外裂简是一页从世界之外坠落的残卷。它不是普通法宝,而像一段残缺的“天道接口”。

它有三种能力:

| 能力 | 表现 | 游戏化 |
|---|---|---|
| 记述 | 把人物、战斗、关系写成本命书 | 战斗叙事、章节生成 |
| 推演 | 让修士借天外灵脉推演招式 | LLM 模型能力差异 |
| 篡改 | 改写人对旧事的记忆 | 羁绊线、月隐名册、书火案 |

玩家的 API Key 在世界观中叫:

```text
本命灵钥
```

它不是门票,而是玩家能否接入本门“天外灵脉”的凭证。  
因此要求玩家上传自己的 API Key 也可以被包装成:

```text
没有本命灵钥,就无法入门。
每个门派只能接通对应灵脉。
验证灵钥,就是试灵根。
```

### 1.2 白石渡裂简夜

十七年前,白石渡出现一场异象:

- 青澜古碑夜半鸣响。
- 沧澜剑冢的旧剑全部出鞘一寸。
- 天机机关锁自行打开。
- 玄机观星盘逆转三息。
- 青冥旧藏书楼灰烬复燃。
- 月隐名册多出一页无名空白。

五宗赶到时,只找到三样东西:

| 遗物 | 含义 |
|---|---|
| 空白本命书 | 记不住过去,但能记下未来 |
| 半枚青澜古碑拓片 | 玩家与古碑旧案有关 |
| 一缕月隐封忆术残痕 | 有人救过玩家,也有人删过玩家记忆 |

玩家就是那一夜唯一被确认活下来的孩子。

### 1.3 五宗为什么都在意玩家

| 门派 | 看见的真相 | 想从玩家身上得到什么 |
|---|---|---|
| 沧澜剑派 | 玩家身上有林泽旧案剑痕 | 查清叛徒林泽是否真叛 |
| 天机阁 | 玩家能打开失控机关锁 | 修复天外裂简的接口 |
| 玄机宗 | 玩家命格是推演空白 | 证明世上有不可计算之人 |
| 青冥派 | 玩家记忆里藏着书火案证词 | 翻出百年前藏书楼真凶 |
| 月隐宫 | 玩家名字出现在月下名册缺页 | 还原被篡改的记忆 |

这个设定能保证玩家选任何门派都合理,同时其它门派仍有理由继续靠近、试探或拉拢玩家。

## 2. 五种入门角色背景

### 2.1 沧澜剑派出身: 被掌刑堂收养的旧案遗孤

玩家被沧澜剑派带回山门,官方说法是:

```text
白石渡旧案中幸存的凡童,剑骨尚可,可入外门修行。
```

真实情况:

- 玩家身上有林泽剑意残痕。
- 沧澜掌刑堂怀疑玩家父母与林泽叛逃有关。
- 陈墨痕把玩家当作师弟照看,也把玩家当作旧案线索。
- 月隐宫叶照眠曾在暗中救过玩家,但沧澜不愿承认这份人情。

初始关系:

| 角色 | 关系 | 初始状态 |
|---|---|---|
| 陈墨痕 `canglan_09` | 严厉师兄/半个师父 | 训练玩家,也监视玩家 |
| 墨听潮 `canglan_01` | 同案遗孤/好友 | 两人都被林泽案改变命运 |
| 叶照眠 `yueyin_11` | 暗中保护者 | 玩家梦里常见他递来一盏灯 |
| 周虚空 `xuanji_09` | 旧案知情者 | 他推演过玩家的命,却没说全 |

玩家核心矛盾:

```text
我是沧澜弟子,还是林泽旧案留下的证物?
```

### 2.2 天机阁出身: 被机关锁认主的修行学徒

玩家被天机阁带走,原因是白石渡遗物中的天机锁芯只对玩家有反应。

表面身份:

```text
外门机关学徒,负责修复旧锁、记录灵脉波动。
```

真实情况:

- 玩家能打开天机阁打不开的月隐名册机关锁。
- 天机阁担心玩家被月隐宫抢回。
- 唐灵枢把玩家当朋友,总拉玩家去闯祸。
- 程星阑发现玩家梦里有一段与月隐冷星沉相同的封存记忆。

初始关系:

| 角色 | 关系 | 初始状态 |
|---|---|---|
| 唐灵枢 `tianji_03` | 损友/冒险搭档 | 带玩家偷看机关库 |
| 鲁照衡 `tianji_02` | 同门工匠 | 帮玩家修本命书书扣 |
| 程星阑 `tianji_17` | 记忆共鸣者 | 怀疑玩家曾见过名册真页 |
| 许藏月 `yueyin_12` | 名册守门人 | 认为天机阁偷走了不该碰的人 |

玩家核心矛盾:

```text
我是在学机关,还是被天机阁当成钥匙?
```

### 2.3 玄机宗出身: 无法被推演的破局者

玄机宗收玩家入门,是因为玩家命格在幻方中显示为空。

表面身份:

```text
外门推演弟子,擅长在死局中走出不合常理的一步。
```

真实情况:

- 周虚空十七年前就算到玩家会活下来。
- 他晚报了三日,导致林泽案与玩家身世被改写。
- 韩极简把玩家当成“证明情感也能破局”的样本。
- 天机阁冯照矩与韩极简相爱,玩家会被卷入两派反对的情感线。

初始关系:

| 角色 | 关系 | 初始状态 |
|---|---|---|
| 韩极简 `xuanji_03` | 同门好友/理性派挚友 | 常用题目表达关心 |
| 周虚空 `xuanji_09` | 可疑师兄/半师半敌 | 知道玩家空白命格来源 |
| 明素问 `xuanji_28` | 后期验忆导师 | 能判断玩家记忆真假 |
| 冯照矩 `tianji_26` | 外宗牵连者 | 因韩极简接近玩家 |

玩家核心矛盾:

```text
我是不被命运束缚的人,还是别人故意留在棋盘上的变数?
```

### 2.4 青冥派出身: 被旧藏书楼收留的失忆书童

青冥派收玩家入门,是因为玩家被发现时抱着半片古碑拓本,拓本背后有青冥旧藏书楼的灰。

表面身份:

```text
藏书楼外门弟子,负责抄录残卷、照看经阁。
```

真实情况:

- 玩家在失忆前可能见过书火案真凶。
- 石听雨曾救过玩家一次,是最早把玩家当普通人而非线索的人。
- 文问松教玩家读经,也希望玩家不要被门派争斗吞掉。
- 月隐宫陆回雪的证词与玩家残缺记忆能互相印证。

初始关系:

| 角色 | 关系 | 初始状态 |
|---|---|---|
| 石听雨 `qingming_02` | 救命恩人/温柔好友 | 玩家低血时容易触发她的关照事件 |
| 文问松 `qingming_12` | 读经师兄/师友 | 引导玩家理解五宗旧盟 |
| 陈千古 `qingming_05` | 同门调查搭档 | 坚信月隐不是书火案真凶 |
| 陆回雪 `yueyin_14` | 旧案目击者 | 她的证词需要玩家记忆补全 |

玩家核心矛盾:

```text
青冥教我求真,但真相可能会伤害青冥自己。
```

### 2.5 月隐宫出身: 被名册藏起来的人

月隐宫收玩家入门,不是因为玩家适合暗杀,而是因为玩家名字曾出现在月下名册缺页。

表面身份:

```text
月隐宫外门弟子,修记忆、潜行和暗线传书。
```

真实情况:

- 玩家当年被月隐宫救过,之后被封去一段记忆。
- 叶照眠知道玩家为何必须被藏起来。
- 顾长忆能逐步恢复玩家旧忆,但每恢复一次都会牵出新敌人。
- 沧澜掌刑堂一直认为月隐宫偷走了重要证人。

初始关系:

| 角色 | 关系 | 初始状态 |
|---|---|---|
| 叶照眠 `yueyin_11` | 暗护者/旧债承担者 | 对玩家温柔,但从不说全真相 |
| 顾长忆 `yueyin_24` | 记忆导师 | 负责修复玩家错乱记忆 |
| 沈鸦青 `yueyin_10` | 危险同门/潜在恋爱线 | 看穿玩家想逃避真相 |
| 闻人守墨 `canglan_33` | 沧澜掌刑压力 | 认为玩家应回沧澜受审 |

玩家核心矛盾:

```text
月隐救了我,也删过我。我该信救命恩,还是信自己的空白记忆?
```

## 3. 固定羁绊角色池

不建议把 200 名弟子都推给玩家。普通用户记不住。  
玩家固定羁绊首批只保留 12 人:

| 角色 | 门派 | 羁绊类型 | 用途 |
|---|---|---|---|
| 墨听潮 `canglan_01` | 沧澜 | 友情/同案遗孤 | 新手期最容易记住的朋友 |
| 陈墨痕 `canglan_09` | 沧澜 | 师兄/严厉导师 | 训练、旧案压力、复仇线 |
| 顾千澜 `canglan_13` | 沧澜 | 好友/情敌/问剑主持 | 三角恋另一端,不能写成坏人 |
| 唐灵枢 `tianji_03` | 天机 | 损友/冒险搭档 | 轻松感、机关线、玩家日常陪伴 |
| 程星阑 `tianji_17` | 天机 | 记忆共鸣 | 月隐名册与玩家记忆线 |
| 韩极简 `xuanji_03` | 玄机 | 理性好友 | 推演、选择与克制 |
| 周虚空 `xuanji_09` | 玄机 | 半师半敌 | 他知道玩家命格真相 |
| 石听雨 `qingming_02` | 青冥 | 救命恩/温柔好友 | 治愈线,可发展为安静恋情 |
| 文问松 `qingming_12` | 青冥 | 师友/礼法导师 | 解释五宗旧盟和门派政治 |
| 沈鸦青 `yueyin_10` | 月隐 | 潜在恋爱/危险盟友 | 主爱情线之一 |
| 叶照眠 `yueyin_11` | 月隐 | 暗中保护者 | 玩家旧命的守护者 |
| 顾长忆 `yueyin_24` | 月隐 | 记忆导师 | 推动主角身世揭示 |

每个门派开局只显示 3-4 个熟人,其余通过下一轮、地图 NPC、战斗章节慢慢出现。  
这样既有复杂关系网,又不会在开局把玩家淹没。

## 4. 玩家固定羁绊故事线

### 4.1 友情线: 墨听潮 · 两个被旧案剩下的人

关键词:

```text
同案遗孤、互相扶持、林泽旧案、友情可能被真相撕裂
```

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 入门后 1-3 轮 | 墨听潮听说玩家也来自白石渡,主动送来一枚旧剑穗。 |
| 2 | 玩家 Lv.5 | 两人在问剑台切磋,发现彼此梦里都有同一场火。 |
| 3 | 玩家第一次击败精英怪 | 墨听潮救场或被玩家救下,友情大幅提升。 |
| 4 | 林泽旧案线出现 | 陈墨痕告诉玩家,墨听潮的亲人可能死于玩家父母之手。 |
| 5 | 玩家选择隐瞒或告知 | 隐瞒会短期保住友情,告知会让关系下降但打开真相线。 |
| 6 | 筑基后 | 墨听潮偷听到叶照眠暗护玩家,误会玩家投向月隐。 |
| 7 | 金丹前后 | 玩家可以与墨听潮结为生死之交,或因旧案成为长期竞争对手。 |

设计价值:

- 这是玩家最容易理解的友情线。
- 不靠复杂人名,只靠“我们都被同一件旧案毁过”建立记忆点。
- 后期友情能转成信任、竞争或误会,有连续张力。

### 4.2 师兄线: 陈墨痕 · 教你挥剑的人也在审你

关键词:

```text
严厉师兄、复仇、训练、监视、父辈真相
```

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 沧澜或跨宗遇见 | 陈墨痕纠正玩家招式,语气很冷,但留下一瓶伤药。 |
| 2 | 玩家连续战败/低血 | 他出现训斥玩家,同时教一招保命剑诀。 |
| 3 | 玩家 Lv.10 破境前 | 他要求玩家参与林泽旧案调查,作为筑基条件之一。 |
| 4 | 玩家帮月隐人物 | 陈墨痕怀疑玩家被月隐利用,关系下降。 |
| 5 | 找到林泽遗令 | 他第一次动摇: 林泽可能不是单纯叛徒。 |
| 6 | 玩家选择公开遗令 | 陈墨痕会站在玩家身边,但与沧澜掌刑堂决裂。 |
| 7 | 元婴前后 | 他可成为玩家固定护道者,或因复仇执念走向对立。 |

设计价值:

- 师徒感不是普通“师父送资源”,而是带有审判和保护的复杂关系。
- 玩家会想知道这个冷硬的人到底信不信自己。

### 4.3 轻松友情线: 唐灵枢 · 把严肃修仙弄出麻烦的人

关键词:

```text
损友、机关、冒险、闯祸、日常感
```

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 下一轮随机 | 唐灵枢误把玩家名字写进天机阁试验名单。 |
| 2 | 玩家去中立市集 | 她拉玩家试用机关飞鸢,结果坠到青冥书车上。 |
| 3 | 玩家与天机好感达到 20 | 她透露“天机锁芯只认玩家”的秘密。 |
| 4 | 黑卷商人线出现 | 她的机关手稿被偷,玩家帮她找回。 |
| 5 | 玩家选择包庇她 | 她成为固定熟人,每隔数轮送情报或机关小道具。 |
| 6 | 金丹后 | 她不再只是搞笑角色,会主动承担一次危险潜入。 |

设计价值:

- 给沉重主线降压。
- 让玩家点击下一轮时期待“今天唐灵枢又闯什么祸”。

### 4.4 爱情/三角线: 沈鸦青、顾千澜与玩家 · 月下问剑

关键词:

```text
三角恋、克制、误会、跨宗压力、每个人都不低级
```

角色关系:

- 顾千澜暗恋沈鸦青,但一直没有说出口。
- 沈鸦青知道顾千澜的心意,却不愿被任何宗门关系定义。
- 玩家如果接近沈鸦青,不是简单抢人,而是卷入“月隐是否该被沧澜原谅”的政治裂缝。

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 第一次在下一轮看见沈鸦青 | 她在月下渡口截住玩家,问玩家是否真想知道过去。 |
| 2 | 顾千澜出现 | 顾千澜替玩家解围,但看沈鸦青的眼神被玩家察觉。 |
| 3 | 玩家与沈鸦青好感 30 | 沈鸦青教玩家识破一次记忆幻术。 |
| 4 | 玩家与顾千澜好感 30 | 顾千澜邀请玩家参加五宗问剑,并坦白自己喜欢沈鸦青。 |
| 5 | 三人同场 | 黑卷商人利用三人关系制造误会: 玩家被传私会月隐。 |
| 6 | 玩家选择 | 支持顾千澜、追求沈鸦青、保持三人同盟、或疏远两人。 |
| 7 | 筑基/金丹破境 | 沈鸦青为救玩家暴露月隐暗线,顾千澜必须决定是否为她作证。 |
| 8 | 中后期 | 三人可以走向道侣、知己、遗憾错过或并肩同盟。 |

设计底线:

- 不写狗血抢夺,写“每个人都有尊严的情感选择”。
- 顾千澜不能降智黑化。他可以嫉妒,但更重要的是他也有宗门责任。
- 沈鸦青不是奖励品,她的选择取决于玩家是否尊重她的秘密和自由。

设计价值:

- 玩家愿意继续点下一轮,因为想知道三人下一次同场会怎样。
- 爱情线同时牵动沧澜与月隐旧怨,不是孤立恋爱小游戏。

### 4.5 治愈/安静恋情线: 石听雨 · 她先把你当普通人

关键词:

```text
救命恩、温柔、药篮、普通生活、与宏大阴谋的反差
```

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 玩家低血进入下一轮 | 石听雨送药,没有问玩家身世。 |
| 2 | 玩家疲劳过高 | 她劝玩家休息,触发一段很短的日常对话。 |
| 3 | 玩家与青冥好感 20 | 她请求玩家帮忙护送药材,不是为了大案,只是为了救外门弟子。 |
| 4 | 书火案推进 | 玩家发现她师门有人参与隐瞒旧案。 |
| 5 | 玩家告诉她真相 | 她会痛苦,但不会逃避,决定和玩家一起查。 |
| 6 | 玩家选择她路线 | 可发展为道侣、挚友或医修同盟。 |

设计价值:

- 给玩家一个不被“钥匙/证据/棋子”定义的关系。
- 与沈鸦青的危险爱情线形成对照: 一个像夜色,一个像灯火。

### 4.6 跨宗恋人线: 韩极简与冯照矩 · 玩家不是主角,但能改变他们

关键词:

```text
旁观者、传信、禁忌恋、理性与机关、成全或拆散
```

这条线重点不是让玩家抢恋爱对象,而是让玩家感觉世界里的人也有自己的欲望。

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 玩家遇见韩极简 | 他请玩家把一道没有署名的题交给天机阁。 |
| 2 | 玩家遇见冯照矩 | 她看懂题目,但只回一个机关零件。 |
| 3 | 两派关系紧张 | 玩家被要求交出两人私通信物。 |
| 4 | 玩家选择 | 帮他们隐瞒、劝他们公开、交给师长、或利用信物换利益。 |
| 5 | 五宗问剑 | 两人被迫对战,玩家可以影响赛前选择。 |
| 6 | 后期 | 他们可能私奔、公开结盟、悲剧分离,或成为玩家最可靠的跨宗联系人。 |

设计价值:

- 让江湖不只围着玩家转。
- 玩家每次帮忙或拆台,都会改变两个人和两派关系。

### 4.7 记忆导师线: 顾长忆 · 你真的想记起来吗

关键词:

```text
记忆修复、代价、真相分层、师徒
```

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 月隐接触或玩家梦魇 | 顾长忆告诉玩家: 空白不是失忆,是封印。 |
| 2 | 玩家 Lv.10 | 他恢复第一枚记忆碎片: 白石渡水下有碑文。 |
| 3 | 玩家 Lv.25 | 第二枚碎片: 玩家曾被一个沧澜人追杀,也被一个月隐人救下。 |
| 4 | 玩家完成书火案节点 | 第三枚碎片: 玩家听见过真凶声音,但不记得名字。 |
| 5 | 玩家与顾长忆好感 60 | 他承认自己曾参与封印玩家记忆。 |
| 6 | 玩家选择原谅或追责 | 原谅会解锁更深记忆,追责会让月隐关系紧张但获得真相线索。 |
| 7 | 化神前后 | 玩家决定是否取回完整记忆。完整记忆会带来力量,也会改变所有固定羁绊。 |

设计价值:

- 这是主角身世揭示的主轴。
- 顾长忆既像老师,又像曾经伤害过玩家的人,关系有复杂度。

### 4.8 半师半敌线: 周虚空 · 他晚报了三日

关键词:

```text
推演、隐瞒、救人还是布局、反派感但可洗白
```

阶段:

| 阶段 | 触发 | 剧情 |
|---:|---|---|
| 1 | 玄机接触 | 周虚空第一次见玩家,说“你本不该在此局”。 |
| 2 | 玩家完成一次越级挑战 | 他承认自己算不出玩家下一步。 |
| 3 | 林泽案推进 | 陈墨痕指出: 周虚空当年晚报三日,才让真凶逃走。 |
| 4 | 玩家质问周虚空 | 他反问玩家: 如果早报,你当年就会被五宗处死。 |
| 5 | 玩家选择相信或追查 | 相信会开启师承线,追查会开启对立线。 |
| 6 | 金丹后 | 周虚空交给玩家一份推演: 玩家未来会害死一个亲近之人。 |
| 7 | 后期 | 玩家可以证明推演错误,或亲手让预言成真。 |

设计价值:

- 给玩家一个“不知道该不该信”的角色。
- 他能让主线有悬疑,也能作为高阶推演玩法的引导者。

## 5. 下一轮羁绊触发规则

当前 `advanceWorld()` 已经会生成大量 NPC 与 NPC 的江湖事件。  
需要新增一层“玩家固定羁绊事件”,让玩家本人被写入关系网。

### 5.1 数据结构建议

```ts
world.player_bonds = {
  [bond_id]: {
    actor_id: string,
    route: 'friendship' | 'romance' | 'mentor' | 'rival' | 'mystery',
    stage: number,
    affinity: number,
    trust: number,
    tension: number,
    flags: string[],
    cooldown: number,
    last_round: number
  }
}
```

推荐初始值:

| 玩家门派 | 初始固定羁绊 |
|---|---|
| 沧澜 | 墨听潮、陈墨痕、叶照眠 |
| 天机 | 唐灵枢、程星阑、许藏月 |
| 玄机 | 韩极简、周虚空、冯照矩 |
| 青冥 | 石听雨、文问松、陆回雪 |
| 月隐 | 叶照眠、顾长忆、沈鸦青 |

后续随着事件解锁,固定羁绊池最多保持 8-10 人。不要把 12 人一次性全塞给玩家。

### 5.2 下一轮触发概率

每次点击下一轮:

```text
基础 18% 触发一条玩家羁绊事件。
如果连续 3 轮没触发,提高到 32%。
如果本轮大事件包含固定羁绊角色,追加 60% 触发后续私聊。
如果玩家刚破境,追加 35% 触发祝贺/试探/邀约。
如果玩家低血或高疲劳,石听雨/陈墨痕/叶照眠等照看型角色权重提高。
```

触发频率建议:

```text
平均 4-6 轮出现一次轻羁绊;
平均 10-15 轮出现一次推进阶段的关键羁绊;
每次大境界突破必定触发 1 条固定羁绊事件。
```

### 5.3 事件类型

| 类型 | 展示标题 | 作用 |
|---|---|---|
| letter | 密信 | 轻量推进,适合下一轮弹出 |
| visit | 来访 | 角色主动找玩家 |
| rescue | 援手 | 低血、战败、危险事件后触发 |
| spar | 切磋 | 提升关系或制造竞争 |
| confession | 坦白 | 推进爱情/旧案/师徒线 |
| misunderstanding | 误会 | 三角恋、跨宗冲突 |
| choice | 抉择 | 玩家必须选立场 |
| memory | 记忆碎片 | 主角身世推进 |
| oath | 结义/道侣/师承 | 阶段性关系确定 |

### 5.4 事件展示格式

下一轮弹出时不要长篇小说,先给玩家短而有钩子的摘要:

```text
羁绊 · 月下问剑
沈鸦青在月下渡口拦住你,她说顾千澜今夜也会来。
她问你: “如果真相会让沧澜与月隐再打一场,你还要听吗?”

选择:
1. 让她继续说。
2. 等顾千澜到场。
3. 反问她当年为何救你。
```

关键羁绊事件才进入更长章节。

## 6. 主线推进结构

玩家身世主线分 8 幕,与等级/破境绑定:

| 幕 | 推荐等级 | 主线节点 | 解锁重点人物 |
|---:|---:|---|---|
| 1 | Lv.1-5 | 入门名帖: 五宗都知道你的名字 | 初始 3 名固定羁绊 |
| 2 | Lv.6-10 | 第一枚记忆碎片: 白石渡水下碑文 | 顾长忆/唐灵枢/墨听潮 |
| 3 | 筑基前 | 破境条件牵出旧案遗物 | 陈墨痕/叶照眠 |
| 4 | Lv.11-25 | 五宗问剑: 玩家第一次被公开注意 | 顾千澜/沈鸦青 |
| 5 | 金丹前 | 林泽遗令或书火残页出现 | 周虚空/文问松 |
| 6 | Lv.26-45 | 月隐名册缺页写出玩家幼名 | 顾长忆/明素问 |
| 7 | 元婴前 | 真凶线索指向五宗高层之一 | 韩沧澜/夜阑宫主/智冥小圣 |
| 8 | Lv.46+ | 玩家选择公开真相、重盟五宗或另立山门 | 全固定羁绊结算 |

这样主线不会靠一次性大段设定灌给玩家,而是每破一个阶段给一块拼图。

## 7. 门派选择如何影响羁绊

同一个角色对不同门派玩家的态度应不同。

### 7.1 沈鸦青对玩家

| 玩家门派 | 初始态度 |
|---|---|
| 沧澜 | 警惕又好奇,因为沧澜曾误判月隐 |
| 天机 | 试探,担心天机阁用机关锁控制名册 |
| 玄机 | 可合作,但不喜欢被推演 |
| 青冥 | 尊重,因为青冥有人愿意替月隐翻案 |
| 月隐 | 同门压力更强,她会更直接、更危险 |

### 7.2 陈墨痕对玩家

| 玩家门派 | 初始态度 |
|---|---|
| 沧澜 | 严厉保护 |
| 天机 | 认为玩家被天机利用 |
| 玄机 | 怀疑玩家与周虚空合谋 |
| 青冥 | 相对信任,因为青冥重证据 |
| 月隐 | 高度戒备,但仍会在危急时救玩家 |

### 7.3 石听雨对玩家

| 玩家门派 | 初始态度 |
|---|---|
| 沧澜 | 因韩小澜旧恩,自然亲近 |
| 天机 | 被机关闯祸牵连,关系轻松 |
| 玄机 | 觉得玩家太会算,但愿意照顾 |
| 青冥 | 同门好友,最容易发展为深羁绊 |
| 月隐 | 起初害怕,后来发现玩家并不冷血 |

### 7.4 周虚空对玩家

| 玩家门派 | 初始态度 |
|---|---|
| 沧澜 | 被追问旧案,关系紧张 |
| 天机 | 互相试探推演与机关 |
| 玄机 | 半师半敌,最容易进入师承线 |
| 青冥 | 对玩家求真态度有兴趣 |
| 月隐 | 认为玩家被记忆绑住,试图让玩家破局 |

## 8. 分支但不失控的写法

剧情要能分支,但不能无限膨胀。  
建议每条固定羁绊只有 3 个长期结局:

### 8.1 关系结局类型

| 结局 | 含义 | 适用 |
|---|---|---|
| 深羁绊 | 道侣、生死之交、师徒确认 | 沈鸦青、石听雨、墨听潮、顾长忆 |
| 带裂痕的同盟 | 仍合作,但有旧伤 | 陈墨痕、周虚空、顾千澜 |
| 对立/错过 | 不一定是敌人,但不会再轻易靠近 | 所有羁绊 |

### 8.2 玩家选择不要太多

每个关键事件最多给 3 个选择:

```text
相信 / 追问 / 隐瞒
帮她 / 帮他 / 先查真相
公开 / 私下处理 / 交给师门
靠近 / 保持距离 / 转移话题
```

选择应改变:

- 与角色好感。
- 与相关门派关系。
- 该羁绊线阶段。
- 后续下一轮可触发事件池。

不要让选择只改一句文案。

## 10. 后续实现优先级

建议按 4 步落地:

| 优先级 | 内容 | 理由 |
|---|---|---|
| P0 | 在角色创建/选门派后写入 `origin_profile` | 先让玩家知道自己是谁 |
| P0 | 初始化 3 个玩家固定羁绊 | 下一轮事件不再只看别人热闹 |
| P1 | 下一轮加入 `player_bond_events` | 形成持续探索欲 |
| P1 | 每个大境界突破必触发一条固定羁绊 | 把数值成长和剧情成长绑在一起 |
| P2 | 关键羁绊事件提供 2-3 个选择 | 让玩家关系真的分支 |
| P2 | 在人物志显示“与我关系” | 帮普通用户记住人名 |

最小可行版本:

```text
每个门派 1 段入门背景。
每个门派 3 个初始羁绊。
每条羁绊先做 4 个阶段。
下一轮 20% 概率触发玩家羁绊。
筑基、金丹、元婴破境时必触发一次关键羁绊。
```

## 11. 设计底线

1. 不要一开始让玩家记 200 个名字。
2. 每条关系都必须能用一句话讲清楚。
3. 爱情线不能把角色写成奖励品,要让对方有自己的目标和底线。
4. 三角恋不能靠降智误会推进,要和门派责任、旧案秘密绑定。
5. 师徒线不能只是送资源,应包含保护、隐瞒、审判和成长。
6. 下一轮事件要短,关键节点再展开成长章节。
7. 每个羁绊事件都要改变至少一个状态:好感、门派关系、阶段、旗标或后续事件池。

最终目标:

```text
玩家点下一轮,不是只想看世界随机刷了什么新闻;
而是会想: 墨听潮有没有知道真相?
沈鸦青今晚会不会来?
顾千澜到底还信不信我?
顾长忆下一次会恢复哪段记忆?
周虚空晚报三日到底是在害我,还是救我?
```

这就是让玩家愿意持续探索的情感钩子。
