# 《灵枢笔录》门派副本与四宗统一主线设计

> 日期: 2026-05-27  
> 目标: 让玩家选择不同门派后,进入完全不同的副本主线。副本不是通用刷材料点,而是“另外三个门派的故事线”。玩家最终目标是统一本局四大门派。  
> 关联文档: `LEVEL_GATED_MAIN_STORY_DUNGEON_CAMPAIGN.md`、`PLAYER_VIEW_SECT_DUNGEON_ROUTEBOOK.md`、`PLAYER_BACKGROUND_AND_FIXED_BONDS_DESIGN.md`、`SECT_DISCIPLE_CHARACTER_BIBLE.md`、`GAMEPLAY_REDESIGN_BANNERLORD_CULTIVATION.md`。

> 玩家实际看到的完整路线、每个副本节点、选择、Boss 和盟印结算,见: [《灵枢笔录》玩家视角门派副本路线书](./PLAYER_VIEW_SECT_DUNGEON_ROUTEBOOK.md)。
> 修为卡点、打怪动机、破境心魔与主线副本的融合节奏,见: [《灵枢笔录》等级卡点 × 主线副本 × 小说化留存方案](./LEVEL_GATED_MAIN_STORY_DUNGEON_CAMPAIGN.md)。

## 0. 核心结论

副本系统应从“刷怪副本”改为“门派归盟副本”。

玩家选择一个门派后,本局主线锁定为:

```text
玩家本门 + 另外三个目标门派 = 本局四大门派
```

玩家最终目标不是单纯灭掉其它门派,而是:

```text
查清旧案
通过试炼
获得盟印
处理关键人物关系
让四大门派签下同一份盟约
```

统一四大门派的完成条件:

| 条件 | 说明 |
|---|---|
| 3 枚外宗盟印 | 每个目标门派副本首通后获得 |
| 3 条门派故事线结算 | 每条线有“归盟、压服、和解、交易”至少一种结局 |
| 至少 2 名关键人物支持 | 例如沈鸦青、唐灵枢、墨听潮等 |
| 最终副本“四宗归盟台”通关 | 以玩家本门为核心重建盟约 |

当前项目已有五宗素材。为降低普通用户记忆负担,首版副本采用“本局四宗制”:

```text
每个周目只让玩家处理 3 个外宗副本。
未进入本局四宗的第五宗,作为幕后观察者、裁判或后续赛季扩展。
```

这样既保留现有五宗资产,又满足“选择一个门派后,打另外三个门派故事线”的目标。

## 1. 副本定位

### 1.1 副本不是经验包

门派副本必须回答玩家三个问题:

```text
我为什么要打这个副本?
我打完以后这个门派会发生什么变化?
这个副本和最终统一四大门派有什么关系?
```

因此每个门派副本都必须包含:

| 模块 | 作用 |
|---|---|
| 门派旧案 | 给副本动机,不是无脑开打 |
| 关键人物 | 让玩家记住门派,不是记一堆杂名 |
| 战斗试炼 | 检验战力,形成爽点 |
| 关系抉择 | 改变该门派对玩家的态度 |
| 盟印奖励 | 作为统一四宗的进度道具 |
| 世界影响 | 改变下一轮事件、地图据点和 NPC 态度 |

### 1.2 副本类型

| 类型 | 是否一次性 | 作用 |
|---|---|---|
| 正传副本 | 首通一次 | 推进门派故事线,获得盟印 |
| 回响副本 | 可重复 | 首通后开放,刷材料和关系补偿 |
| 高难问心 | 可挑战 | 对应门派高难试炼,给称号、法宝和特殊结局 |
| 最终归盟副本 | 一次性主线 | 集齐三枚盟印后开启,统一四宗 |

正传副本一定要有剧情和选择。  
回响副本才承担日常材料产出。

## 2. 副本通用结构

每个外宗副本采用 5 节结构:

| 节点 | 内容 | 玩家体验 |
|---|---|---|
| 1. 入境 | 进入目标门派地界,遇到第一个关键人物 | 知道这条线在讲什么 |
| 2. 旧案 | 发现该门派与白石渡、裂简或本门旧怨有关 | 产生继续打的理由 |
| 3. 试炼 | 3-5 个战斗/机关/辩经节点 | 获得副本爽感 |
| 4. 问心 | 掌门、真传或守关人给最终抉择 | 关系分支 |
| 5. 归盟 | 获得盟印或压服该门派 | 统一进度前进 |

副本内节点建议:

```text
普通战斗 2 个
精英战斗 1 个
剧情选择 2 个
关键人物对话 2 个
最终 Boss/试炼 1 个
局末总结章 1 个
```

默认不要每个小节点都调用高成本长文。  
只有第 1 次进入、最终 Boss、归盟结算使用高质量叙事。

## 3. 统一四宗的阶段节奏

| 阶段 | 推荐等级 | 主线目标 | 副本 |
|---|---:|---|---|
| 第一阶段 | Lv.1-10 | 入门与熟人建立 | 无大型副本,只开序章小试炼 |
| 第二阶段 | Lv.11-25 筑基 | 争取第一个外宗 | 第一个门派正传副本 |
| 第三阶段 | Lv.26-45 金丹 | 处理第二个外宗旧案 | 第二个门派正传副本 |
| 第四阶段 | Lv.46-70 元婴 | 压服或和解第三个外宗 | 第三个门派正传副本 |
| 第五阶段 | Lv.71+ 化神 | 四宗归盟 | 最终副本“四宗归盟台” |

这样玩家目标会非常清楚:

```text
筑基后拉第一个盟友。
金丹后解决第二个门派。
元婴后集齐三枚盟印。
化神后统一四宗。
```

## 4. 路线总表

### 4.1 本局四宗选择规则

每个玩家门派都有 3 个目标门派副本和 1 个幕后门派。

| 玩家门派 | 目标门派副本 1 | 目标门派副本 2 | 目标门派副本 3 | 幕后门派 |
|---|---|---|---|---|
| 沧澜剑派 | 天机阁 | 青冥派 | 月隐宫 | 玄机宗 |
| 天机阁 | 沧澜剑派 | 玄机宗 | 青冥派 | 月隐宫 |
| 玄机宗 | 天机阁 | 沧澜剑派 | 月隐宫 | 青冥派 |
| 青冥派 | 沧澜剑派 | 天机阁 | 月隐宫 | 玄机宗 |
| 月隐宫 | 沧澜剑派 | 青冥派 | 天机阁 | 玄机宗 |

幕后门派不进入首版三个归盟副本,但会:

- 在下一轮中制造传闻。
- 在关键副本里派观察者。
- 在最终归盟台前给玩家一份“真假难辨”的情报。
- 后续可作为二周目或隐藏真结局。

## 5. 沧澜玩家路线

玩家身份:

```text
沧澜掌刑堂收养的旧案遗孤。
目标: 让剑派从复仇走向重盟。
```

### 5.1 天机阁副本: 《机关廊锁心案》

副本主题:

```text
天机阁的机关锁能打开月隐名册,但锁芯只响应玩家。
```

关键人物:

| 角色 | 作用 |
|---|---|
| 唐灵枢 `tianji_03` | 损友,带玩家闯机关廊 |
| 鲁照衡 `tianji_02` | 修锁工匠,帮助玩家理解锁芯 |
| 罗开阖 `tianji_21` | 名册机关锁看守者 |
| 谷开明 `tianji_29` | 主张公开机关档案 |

节点:

| 节点 | 内容 |
|---|---|
| 入境 | 唐灵枢把玩家误登记成“锁芯试验者” |
| 旧案 | 机关锁记录显示,白石渡夜有人提前开过锁 |
| 试炼 | 机关廊三层,每层选择不同机关祝福 |
| 问心 | 罗开阖要求玩家交出本命书书扣 |
| 归盟 | 玩家选择公开锁芯记录或让天机阁保管副本 |

最终 Boss/试炼:

```text
万象锁阵
```

结局:

| 选择 | 结果 |
|---|---|
| 公开锁芯记录 | 天机盟印到手,天机守旧派不满 |
| 交给谷开明 | 天机公开派支持玩家,但机关锁暂时失控 |
| 私藏书扣 | 玩家获得特殊法宝,天机好感下降 |

奖励:

- 天机盟印。
- 机关类祝福解锁。
- 唐灵枢进入固定熟人池。
- 下一轮增加“机关传闻”事件。

### 5.2 青冥派副本: 《旧藏书楼书火案》

副本主题:

```text
青冥旧藏书楼百年前失火,月隐宫背锅。玩家残缺记忆中可能有真凶声音。
```

关键人物:

| 角色 | 作用 |
|---|---|
| 石听雨 `qingming_02` | 救命恩人,医修线 |
| 文问松 `qingming_12` | 读经师兄,解释旧盟 |
| 陈千古 `qingming_05` | 坚信月隐无罪 |
| 陆正辞 `qingming_33` | 当年审案者之一 |

节点:

| 节点 | 内容 |
|---|---|
| 入境 | 石听雨在雨夜救治受伤外门弟子 |
| 旧案 | 玩家发现书火灰烬与白石渡拓片同源 |
| 试炼 | 辩经、护书、战守卷灵三个节点 |
| 问心 | 陆正辞要求玩家只交证据,不得公开猜测 |
| 归盟 | 玩家决定是否替月隐翻案 |

最终 Boss/试炼:

```text
守卷心魔
```

结局:

| 选择 | 结果 |
|---|---|
| 公开月隐无罪线索 | 青冥年轻派支持,守旧派敌意上升 |
| 暂时保密继续查 | 石听雨和文问松信任增加 |
| 只为沧澜取盟印 | 青冥盟印到手,但后续书火真相线变难 |

奖励:

- 青冥盟印。
- 经卷类被动祝福。
- 石听雨羁绊推进。
- 开启“书火残页”收集。

### 5.3 月隐宫副本: 《月下名册缺页》

副本主题:

```text
玩家名字在月下名册缺页上,月隐宫既救过玩家,也删过玩家记忆。
```

关键人物:

| 角色 | 作用 |
|---|---|
| 沈鸦青 `yueyin_10` | 爱情/危险盟友线 |
| 叶照眠 `yueyin_11` | 暗中保护者 |
| 顾长忆 `yueyin_24` | 记忆导师 |
| 夜阑宫主 `yueyin_37` | 掌握书火案真相 |

节点:

| 节点 | 内容 |
|---|---|
| 入境 | 沈鸦青在月下渡口截住玩家 |
| 旧案 | 顾长忆恢复第一枚记忆碎片 |
| 试炼 | 潜行、辨忆、影战三节点 |
| 问心 | 夜阑宫主问玩家是否愿意承认月隐救命恩 |
| 归盟 | 玩家决定是否让沧澜公开旧错 |

最终 Boss/试炼:

```text
无名影册
```

结局:

| 选择 | 结果 |
|---|---|
| 为月隐作证 | 月隐盟印到手,沧澜掌刑堂压力上升 |
| 要求月隐交出完整名册 | 顾长忆信任下降,但获得关键证据 |
| 与沈鸦青私下结盟 | 解锁三角恋后续,月隐归盟条件暂缓 |

奖励:

- 月隐盟印。
- 记忆碎片。
- 沈鸦青/顾长忆羁绊推进。
- 最终副本可选择“赎罪归盟”路线。

### 5.4 幕后门派: 玄机宗

玄机宗不作为沧澜路线的首版副本目标,但周虚空会持续出现。

作用:

- 在每个副本后给玩家一句推演。
- 暗示林泽旧案不是单纯叛逃。
- 最终归盟台前揭露“三日不报”的真相。

## 6. 天机玩家路线

玩家身份:

```text
被机关锁认主的天机学徒。
目标: 证明机关不是控制五宗的工具,而是重建盟约的钥匙。
```

| 副本 | 主题 | 关键人物 | 盟印条件 |
|---|---|---|---|
| 沧澜剑派《剑冢林泽案》 | 天机曾送林泽出城,沧澜怀疑天机包庇叛徒 | 陈墨痕、墨听潮、萧问剑 | 证明林泽遗令不是天机伪造 |
| 玄机宗《无字天衍图》 | 天机与玄机同源异算,半卷天衍图互相指责 | 韩极简、周虚空、明素问 | 赢下推演局,或承认机关也有盲区 |
| 青冥派《青澜古碑注疏》 | 青冥认为天机篡改古碑注疏 | 石听雨、文问松、宋扶碑 | 修复古碑注疏,不强占解释权 |
| 幕后月隐 | 名册机关锁的真正主人 | 许藏月、顾长忆 | 最终前揭示锁芯来源 |

### 6.1 天机路线差异

同样是沧澜副本,天机玩家看到的是:

```text
我门派可能真的帮过林泽。
如果我公开机关记录,天机会背锅。
如果我隐藏记录,沧澜永远不会归盟。
```

天机路线的副本选择更偏“公开/保密/技术中立”。

## 7. 玄机玩家路线

玩家身份:

```text
无法被推演的破局者。
目标: 证明统一四宗不是唯一答案,但仍要让四宗停止互相消耗。
```

| 副本 | 主题 | 关键人物 | 盟印条件 |
|---|---|---|---|
| 天机阁《双算天衍局》 | 天机机关与玄机推演谁更接近真相 | 唐灵枢、冯照矩、赵星珩 | 解开天衍图半卷,不伤韩极简与冯照矩 |
| 沧澜剑派《三日不报旧案》 | 周虚空晚报三日导致沧澜恨玄机 | 陈墨痕、墨听潮、韩沧澜 | 解释三日真相,承受沧澜问剑 |
| 月隐宫《名册逆推》 | 月隐记忆与玄机推演互相矛盾 | 沈鸦青、顾长忆、叶千忆 | 证明名册不是伪造,或指出被篡改页 |
| 幕后青冥 | 用经义审判推演伦理 | 文问松、陈照古 | 最终前提出归盟礼法 |

### 7.1 玄机路线差异

玄机玩家不是靠武力压服其它门派,而是靠:

- 推演出旧案关键缺口。
- 承认推演曾经伤害别人。
- 在最终归盟时给四宗一个“不会被任何一宗独占”的制度。

## 8. 青冥玩家路线

玩家身份:

```text
被旧藏书楼收留的失忆书童。
目标: 以证据和礼法统一四宗,但真相会伤害本门。
```

| 副本 | 主题 | 关键人物 | 盟印条件 |
|---|---|---|---|
| 沧澜剑派《青澜古碑护碑》 | 沧澜与青冥共守古碑,但碑文被改过 | 玄沧明、沈砚白、韩沧澜 | 找到古碑第三层刻痕 |
| 天机阁《机关锁与旧藏书楼》 | 天机机关锁可能保存书火案证据 | 唐灵枢、鲁机巧、谷开明 | 取得机关锁旧档,不让天机背全锅 |
| 月隐宫《书火翻案》 | 月隐背锅百年,但青冥守旧派不愿认错 | 沈鸦青、陆回雪、夜阑宫主 | 让青冥承认至少一部分误判 |
| 幕后玄机 | 判断证词真假 | 明素问、周虚空 | 最终前验证关键记忆 |

### 8.1 青冥路线差异

青冥玩家最大的痛点:

```text
如果真相证明青冥错了,我还要不要公开?
```

因此青冥副本选择更偏“证据/礼法/公开代价”。

被名册藏起来的人。
目标: 让四宗承认月隐不是只会潜伏和背叛的影子。

月隐路线应更强调潜入、证据、选择是否公开身份。

## 10. 最终副本: 四宗归盟台

开启条件:

```text
完成 3 个外宗正传副本。
获得 3 枚外宗盟印。
玩家至少达到化神前后,或完成等价主线战力门槛。
```

副本背景:

```text
青澜古碑下方显出第四层碑文。
碑文不是功法,而是一份旧盟约。
四宗祖师曾约定: 若天外裂简再临,四宗不得互相夺简,必须共立执笔者。
但这份盟约被第五只手篡改过。
```

### 10.1 最终副本结构

| 节点 | 内容 |
|---|---|
| 1. 三印入碑 | 玩家把三枚外宗盟印嵌入古碑 |
| 2. 旧怨回响 | 三个已通关副本的关键选择依次回响 |
| 3. 四宗问心 | 每个门派派一名代表质问玩家 |
| 4. 幕后门派现身 | 本局幕后门派揭示它一直在观察玩家 |
| 5. 归盟试炼 | 四宗代表共同进入战斗或辩经 |
| 6. 统一结局 | 根据玩家此前选择生成归盟形式 |

### 10.2 统一不是只有一种

| 统一方式 | 条件 | 结局气质 |
|---|---|---|
| 四宗重盟 | 证据公开、关键人物支持高 | 理想结局 |
| 执笔盟主 | 玩家战力压服,但关系一般 | 霸道结局 |
| 暗盟共治 | 月隐/玄机相关选择偏保密 | 灰色结局 |
| 礼法盟约 | 青冥证据线完成度高 | 稳健结局 |
| 机关盟约 | 天机公开派支持高 | 制度结局 |
| 剑誓盟约 | 沧澜问剑胜场高 | 武力结局 |

失败不应直接 Game Over。  
如果最终归盟失败,进入:

```text
四宗裂盟状态
```

裂盟状态下:

- 门派副本回响难度提高。
- 下一轮更容易触发门派冲突。
- 玩家可以补做关系线,再次挑战归盟台。

## 11. 副本奖励设计

### 11.1 首通奖励

| 奖励 | 作用 |
|---|---|
| 外宗盟印 | 统一四宗主线进度 |
| 外宗祝福 | 副本内/战斗构筑能力 |
| 关键人物入熟人池 | 后续下一轮会持续触发 |
| 门派关系大幅变化 | 影响 NPC 态度、交易、传授 |
| 正传章节 | 写入本命书 |

### 11.2 回响副本奖励

| 副本 | 可刷材料 |
|---|---|
| 沧澜回响 | 剑意残页、剑匣铁、问剑令 |
| 天机回响 | 机关齿、锁芯碎片、星盘针 |
| 玄机回响 | 幻方砂、推演符、无字残页 |
| 青冥回响 | 经卷残页、药草、古碑拓墨 |
| 月隐回响 | 月影针、名册碎珠、封忆灯油 |

回响副本不再重复推进主线,只提供:

- 材料。
- 羁绊补偿。
- 高难挑战。
- 少量下一轮事件权重。

## 12. UI 与玩家认知

主城不应显示“副本列表 20 个”。  
应显示一张清晰的统一进度图:

```text
本门: 沧澜剑派
目标: 统一四宗

天机阁  [未归盟]  机关廊锁心案
青冥派  [调查中]  旧藏书楼书火案
月隐宫  [敌意]    月下名册缺页

当前推荐:
先去青冥派,因为你已获得书火残页 2/3。
```

每个门派卡片只显示 4 个核心信息:

| 信息 | 示例 |
|---|---|
| 当前态度 | 敌意、观望、可谈、愿盟 |
| 主线副本 | 旧藏书楼书火案 |
| 关键人物 | 石听雨、文问松 |
| 归盟条件 | 书火残页 2/3,击败守卷心魔 |

用户不需要先记住所有人名,只要知道:

```text
我要拉拢哪个门派。
我还差什么条件。
谁是这条线里最重要的人。
```

## 14. 落地优先级

| 优先级 | 内容 | 理由 |
|---|---|---|
| P0 | 按玩家门派生成 `unification.route` | 先确定本局要统一哪四宗 |
| P0 | 做 5 条路线表,每条 3 个目标副本 | 让不同门派开局真的不一样 |
| P1 | 每个副本先做 5 节文字节点 + 1 个战斗 Boss | 最小闭环可玩 |
| P1 | 首通给盟印,更新门派态度 | 让副本改变世界状态 |
| P1 | 主城增加“四宗归盟进度” | 玩家知道长期目标 |
| P2 | 副本选择影响固定羁绊 | 把副本和人物线绑起来 |
| P2 | 做回响副本材料刷取 | 首通后可重复玩 |
| P3 | 最终副本“四宗归盟台”多结局 | 完成长期目标 |

最小可行版本:

```text
玩家选门派后,生成 3 个目标门派副本。
每个副本 5 个节点,其中 2 个战斗、2 个剧情选择、1 个 Boss。
打完给该门派盟印。
集齐三枚盟印,解锁四宗归盟台。
```

## 15. 设计底线

1. 副本不是通用刷怪入口,必须绑定门派故事线。
2. 玩家选不同门派,目标副本、关键人物、旧案解释都要不同。
3. 统一四宗不是灭门,而是归盟、压服、和解、制度或暗盟。
4. 每个副本只推 2-4 个关键人物,避免人名过载。
5. 首通副本必须改变世界状态,否则用户没有持续推进欲望。
6. 回响副本才能承担刷材料,正传副本承担剧情和目标。
7. 最终归盟台要根据前三个副本选择生成不同统一结局。

玩家体验目标:

```text
我不是在刷副本。
我是带着本门立场,去解决另外三个门派的旧案。
每打通一个副本,一个门派就从敌人、观望者或债主,变成我的盟约一部分。
最终,四大门派会因为我的选择统一。
```
