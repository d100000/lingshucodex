## 3. 战斗系统精通

### 3.1 数值公式

**伤害计算**:
```
damage = attacker.atk × card.power × (100 / (100 + defender.def_))
         × (crit ? crit_dmg : 1) × random(0.9, 1.1)
```

**命中率**:
```
hit_rate = card.hit_rate - defender.evasion
```

**暴击率**:
```
crit = random() < attacker.crit_rate
```

### 3.2 卡牌使用策略

#### 通用 4 卡

| 卡牌 | 灵气 | 倍率 | 用法 |
|---|---|---|---|
| 灵气一击 | 8 | 1.0x | 最便宜,稳定输出,任何时候都能用 |
| 凝神咒 | 15 | buff | **下一击 ATK +50%**,蓄力杀招 |
| 寒冰诀 | 25 | 1.5x | 中级输出,但命中率 0.85(低) |
| 灵丹 | 0 | heal | **回 30% HP**,救命神器 |

**组合连招**:`凝神咒 → 寒冰诀(暴击)` = 输出爆炸

#### 沧澜专属

| 卡牌 | 灵气 | 倍率 | 用法 |
|---|---|---|---|
| 剑诀·初 | 12 | 1.2x | 性价比之王,日常主打 |
| 沧澜九式 | 60 | 2.5x | 大招,**配合凝神咒** 5x+ ATK |

#### 天机专属

| 卡牌 | 灵气 | 倍率 | 用法 |
|---|---|---|---|
| 机关连发 | 20 | 1.4x | 稳定 0.92 命中,日常主打 |
| 万象归元 | 50 | 2.0x | AOE 大招,多敌时无敌 |

### 3.3 灵气管理

**最大化战斗效率**:
- 战斗开始:HP 满 / 灵气满
- 节奏:**先用凝神咒,再用大招**(50%++ 加成)
- HP < 30% 时**优先用灵丹**(免费,回 30%)
- 灵气 < 30 时**用最便宜的卡**(灵气一击 8 气)

### 3.4 屏幕反馈解读

战斗中各种闪光的含义:

| 反馈 | 含义 |
|---|---|
| 屏幕震动 | 受击 / 命中 |
| 大幅震动 | **暴击**!伤害翻倍 |
| 全屏金色闪光 | 关键暴击 / 大招 |
| 立绘抖动 | 受击 |
| 数字变金色变大 | 暴击伤害 |
| 全屏绿色闪光 | 战斗胜利 |

## 10. 常见误区

### ❌ 误区 1:升档越快越好
**错。** 升档后单场战斗成本翻倍,如果装备不够强,等于花两倍钱看一样的内容。
**正确**:卡在境界上限,装备攒齐再升档。

### ❌ 误区 2:可以中途换门派
**错。** 选派后必须转世重修(删档),且**新 Key 必须支持新派**。
**正确**:选派前用入门页探测,确认有 2 派以上可选再选定。

### ❌ 误区 3:战斗中刷新页面没事
**错。** 战斗状态在内存,刷新即丢失。
**正确**:战斗中不要刷新,实在要离开点"逃离战斗"。

### ❌ 误区 4:bobdong.cn 不是官方
**错。** bobdong.cn 是 FireworkRouter 网关,**整合了 Claude + GPT 等主流模型**,**「全网源头 token,更真、更快、更强」**,是官方推荐入口。

### ❌ 误区 5:fallback 模板说明 LLM 坏了
**错。** Fallback 是设计中的优雅降级机制,LLM 偶尔失败属正常。
**正确**:看后端日志 `[LLM Fallback]` 字样,如果 >5% 战斗触发,才需要排查。
