# 《灵枢笔录》等级卡点 × 主线副本 × 小说化留存方案

> 日期: 2026-05-27  
> 目标: 把“修为升级卡点”和“门派副本主线”融合起来,让玩家通过打怪提升实力,再挑战主线副本;副本必须像一部长篇修仙小说的主线章节,情感丰富、目标明确、值得持续投入。  
> 关联文档: `CULTIVATION_LEVEL_PROGRESSION_REDESIGN.md`、`SECT_DUNGEON_UNIFICATION_CAMPAIGN_DESIGN.md`、`PLAYER_VIEW_SECT_DUNGEON_ROUTEBOOK.md`、`PLAYER_BACKGROUND_AND_FIXED_BONDS_DESIGN.md`。

## 0. 调研结论: 修仙主线为什么让人追下去

这次外部调研不用于照搬小说情节,而是提炼可转化为游戏主线的结构。

### 0.1 可借鉴结构

| 来源观察 | 可转化为游戏设计 |
|---|---|
| 仙侠/修仙类型通常围绕修炼、神话、道佛儒、武术、炼丹等元素展开,并天然适合“等级成长”叙事。 | 游戏主线必须把等级、破境、宗门关系和法宝资源绑成同一条线。 |
| 《凡人修仙传》这类“凡人流”早期吸引力在于低阶弟子为了筑基机会进入残酷试炼,靠谨慎、资源和机缘活下来。 | 筑基前后的第一个大副本应是“为了破境不得不去”的生死试炼。 |
| 许多长篇升级小说采用“升级 + 换地图”结构,每次境界提升都进入更大的地图和更复杂的势力冲突。 | 每个大境界都应解锁一个新的外宗副本,而不是只提高数值。 |
| 经典升级文常有人格化导师或引路人,负责给任务、开眼界、提供修炼目标。 | 顾长忆、陈墨痕、唐灵枢、周虚空等固定羁绊角色要承担“剧情导师”功能。 |
| 修仙小说的天劫与心魔最适合做情感高潮: 外在雷劫考验战力,内在心劫考验玩家过去的选择。 | 每个破境卡点不只要修为,还要一次“问心/心魔”副本节点。 |
| 《一念永恒》中“四宗合并为逆河宗”的宗门格局变化说明,宗门统一本身可以成为长期主线目标。 | 本作的最终目标“统一四宗”是合理的修仙长线目标,但必须由人物关系和旧案推动。 |

### 0.2 对《灵枢笔录》的核心启发

玩家持续打怪不是为了数字本身,而是为了这句话:

```text
我还差一点修为、一个关键材料、一段人物信任,就能进入下一章主线副本。
```

因此每个等级卡点都要绑定三件事:

| 卡点要素 | 作用 |
|---|---|
| 修为门槛 | 玩家通过打怪、精英怪、悬赏、回响副本积累 |
| 剧情副本 | 推进主线,拿盟印或记忆碎片 |
| 情感选择 | 让玩家觉得自己不是刷关,而是在改变人物命运 |

## 1. 总体主线结构

玩家每个周目采用:

```text
本门序章
-> 第一外宗副本
-> 第二外宗副本
-> 第三外宗副本
-> 四宗归盟台
```

对应等级:

| 等级段 | 境界 | 主线章节 | 玩家目标 | 打怪意义 |
|---:|---|---|---|---|
| Lv.1-10 | 炼气 | 本门序章: 白石渡残梦 | 证明自己有资格入门,解开第一枚记忆碎片 | 打低阶妖兽,熟悉战斗,收集入门信物 |
| Lv.10->11 | 筑基卡点 | 本门问心试炼 | 完成筑基,正式拥有挑战外宗的资格 | 打精英怪攒修为和筑基材料 |
| Lv.11-25 | 筑基 | 第一外宗副本 | 拉拢或压服第一个外宗,拿第一枚盟印 | 打对应门派地界怪物,解锁副本节点 |
| Lv.25->26 | 金丹卡点 | 第一盟印入丹 | 用第一枚盟印凝成金丹道基 | 打副本回响和精英首领补足修为 |
| Lv.26-45 | 金丹 | 第二外宗副本 | 处理更痛的旧案,拿第二枚盟印 | 打中阶妖巢、据点和门派试炼 |
| Lv.45->46 | 元婴卡点 | 元婴照旧人 | 选择保谁、信谁,让某段关系质变 | 打 Boss 和秘境材料,准备元婴心魔 |
| Lv.46-70 | 元婴 | 第三外宗副本 | 解决最危险的宗门旧怨,拿第三枚盟印 | 打高阶据点、回响副本、守关怪 |
| Lv.70->71 | 化神卡点 | 三印化神 | 三枚盟印共鸣,玩家成为四宗可见的核心人物 | 打三宗回响补齐三印共鸣 |
| Lv.71-100 | 化神 | 四宗归盟台 | 统一四宗,决定盟约形式 | 打最终准备战、心魔战、裂简伪盟 |

关键体验:

```text
打怪 -> 得修为/线索/材料 -> 解锁副本下一幕 -> 触发人物情感选择 -> 破境 -> 开新门派线
```

这比“打怪 -> 升级 -> 打更强怪”更有持续动力。

## 2. 每个破境卡点如何融合副本

### 2.1 Lv.1-10 炼气: 本门序章

玩家刚入门时,不能直接扔进外宗副本。  
这一段要让玩家建立三个认知:

```text
我是谁。
本门为什么收我。
我身边哪 2-3 个人值得记住。
```

本门序章统一叫:

```text
白石渡残梦
```

但不同门派看到的解释不同:

| 玩家门派 | 序章情感核心 | 第一个熟人 |
|---|---|---|
| 沧澜 | 我被剑派收养,但他们也在审我 | 墨听潮、陈墨痕 |
| 天机 | 我被机关锁认主,像一把活钥匙 | 唐灵枢、鲁照衡 |
| 玄机 | 我的命格推演为空,师门既惊喜又忌惮 | 韩极简、周虚空 |
| 青冥 | 我在旧藏书楼灰烬中醒来,记忆像缺页 | 石听雨、文问松 |
| 月隐 | 我被月下名册藏起,有人救我也删我 | 叶照眠、顾长忆 |

炼气期打怪目标:

| 打怪内容 | 用途 |
|---|---|
| 低阶妖兽 | 获得基础修为 |
| 本门试炼怪 | 掉落本门入门信物 |
| 首胜怪物图鉴 | 给一次性有效修为奖励 |
| 小型悬赏 | 解锁本门序章下一幕 |

炼气末卡点:

```text
Lv.10 -> Lv.11
修为需求: 77,000
剧情需求: 完成本门问心试炼
材料需求: 筑基丹 + 本门入门信物
情感需求: 初始羁绊角色至少 1 人信任达到“友善”
```

这一关不应该只是点升级,而是第一次小说高潮:

```text
我筑基那夜,白石渡残梦重现。
有人在梦里救我,也有人在梦里追我。
我必须选择醒来后先相信谁。
```

### 2.2 Lv.11-25 筑基: 第一外宗副本

筑基后的第一个外宗副本是“玩家第一次走出本门”。

设计重点:

- 难度不能太高,但必须有正传感。
- 情感重点放在“交朋友/建立第一份外宗信任”。
- 让玩家体验: 我打怪不是为了刷,而是为了进副本下一层。

副本进度卡点:

| 副本阶段 | 推荐等级 | 解锁方式 |
|---|---:|---|
| 入境 | Lv.11 | 筑基完成自动解锁 |
| 起疑 | Lv.13 | 击败 3 个目标地界普通怪 |
| 试炼一 | Lv.15 | 收集 2 个门派线索 |
| 试炼二 | Lv.18 | 击败 1 个精英怪 |
| 问心 | Lv.21 | 与关键人物好感达到 20 或完成支线 |
| 守关 Boss | Lv.24 | 修为接近筑基末段 |
| 归盟结算 | Lv.25 | 击败 Boss,拿第一盟印 |

金丹卡点:

```text
Lv.25 -> Lv.26
修为需求: 270,000
剧情需求: 第一外宗盟印
心魔需求: 你是否愿意把本门秘密告诉第一个外宗盟友
```

这会让玩家为了挑战第一个副本主动去打怪:

```text
我还打不过守关 Boss。
我先去刷地界妖兽、拿门派线索、提升到 Lv.24。
```

### 2.3 Lv.26-45 金丹: 第二外宗副本

金丹期主线要更狠一点。  
这一阶段不是交朋友,而是“真相开始伤人”。

设计重点:

- 第二外宗副本揭露旧案中更痛的一面。
- 玩家必须在两个可信角色之间做选择。
- 打怪目标从普通妖兽升级为妖巢、据点、门派精英试炼。

副本进度卡点:

| 副本阶段 | 推荐等级 | 解锁方式 |
|---|---:|---|
| 入境 | Lv.26 | 金丹成功 |
| 起疑 | Lv.30 | 击败目标地界精英怪或完成悬赏 |
| 试炼一 | Lv.34 | 清理一个小据点 |
| 试炼二 | Lv.38 | 收集 3 个旧案证据 |
| 问心 | Lv.41 | 关键人物好感/敌意达到阈值 |
| 守关 Boss | Lv.44 | 完成副本前置三证 |
| 归盟结算 | Lv.45 | 拿第二盟印 |

元婴卡点:

```text
Lv.45 -> Lv.46
修为需求: 850,000
剧情需求: 第二外宗盟印
心魔需求: 有一个熟人会因你的选择受伤、失踪或与你决裂
```

这里要体现小说感:

```text
主线不是一路正确。
我每往前一步,都会让某个人付出代价。
```

### 2.4 Lv.46-70 元婴: 第三外宗副本

元婴期是“玩家开始成为势力人物”。  
第三外宗副本应该是本周目最危险的外宗线。

设计重点:

- 第三盟印牵涉玩家身世核心。
- 玩家不再只是调查者,而是被所有门派要求表态的人。
- 副本 Boss 应是“旧案怨念 + 门派机制 + 玩家心魔”的混合。

副本进度卡点:

| 副本阶段 | 推荐等级 | 解锁方式 |
|---|---:|---|
| 入境 | Lv.46 | 元婴成功 |
| 起疑 | Lv.50 | 完成一次高阶妖巢 |
| 试炼一 | Lv.55 | 击败门派守关精英 |
| 试炼二 | Lv.60 | 清理资源据点或秘境 |
| 问心 | Lv.64 | 触发一条固定羁绊关键事件 |
| 守关 Boss | Lv.69 | 三条证据线齐备 |
| 归盟结算 | Lv.70 | 拿第三盟印 |

化神卡点:

```text
Lv.70 -> Lv.71
修为需求: 2,100,000
剧情需求: 三枚盟印
心魔需求: 你必须决定四宗统一后,裂简由谁保管
```

### 2.5 Lv.71-100 化神: 四宗归盟台

化神期不再开普通门派副本。  
这里要进入真正的主线终局:

```text
四宗归盟台
```

最终副本不是单纯 Boss 战,而是把前三个副本里的选择都拉回来审判玩家。

节点:

| 节点 | 推荐等级 | 内容 |
|---|---:|---|
| 三印入碑 | Lv.71 | 三枚盟印嵌入青澜古碑 |
| 旧怨回响 | Lv.75 | 前三副本中被你伤过的人出现 |
| 熟人作证 | Lv.80 | 支持你的 2-4 个角色出场 |
| 幕后门派现身 | Lv.85 | 本局第五宗给出隐藏真相 |
| 心魔天劫 | Lv.90 | 玩家面对“统一四宗是否只是另一种独占” |
| 裂简伪盟 | Lv.95 | 最终 Boss |
| 归盟结局 | Lv.100 | 多结局统一 |

化神后继续玩的目标:

- 回响副本高难。
- 四宗裂盟修复。
- 隐藏第五宗真结局。
- 建宗、收徒、灵脉战争。

## 3. 五条玩家门派路线与等级融合

### 3.1 沧澜路线: 从复仇剑到重盟剑

主线情绪:

```text
我从被审判的旧案遗孤,变成愿意替各宗旧错作证的人。
```

| 等级段 | 副本/章节 | 主线内容 | 打怪目标 | 卡点奖励 |
|---:|---|---|---|---|
| Lv.1-10 | 白石渡残梦·沧澜 | 墨听潮成为朋友,陈墨痕训练并监视我 | 山门试炼怪、剑冢残影 | 筑基资格 |
| Lv.11-25 | 天机阁《机关廊锁心案》 | 查出白石渡开锁者并非普通天机弟子 | 机关铜鸢、锁阵守卫 | 天机盟印 |
| Lv.26-45 | 青冥派《旧藏书楼书火案》 | 书火案与白石渡灰烬同源 | 守书灵、经卷心魔 | 青冥盟印 |
| Lv.46-70 | 月隐宫《月下名册缺页》 | 发现月隐救过我也封过我 | 月影迷廊、无名影册 | 月隐盟印 |
| Lv.71-100 | 四宗归盟台 | 沧澜必须承认复仇不能代替真相 | 裂简伪盟、心魔天劫 | 四宗重盟/剑誓盟约 |

小说化高潮:

```text
在月隐宫副本后,陈墨痕质问我是否背弃沧澜。
墨听潮也开始怀疑我是不是早知道他的亲人为何死去。
我若选择为月隐作证,沧澜掌刑堂会视我为叛逆。
但这也是沧澜从复仇走向重盟的唯一机会。
```

### 3.2 天机路线: 从活钥匙到立约者

主线情绪:

```text
我不是天机阁控制四宗的工具。
我要证明机关可以成为防止背叛的制度。
```

| 等级段 | 副本/章节 | 主线内容 | 打怪目标 | 卡点奖励 |
|---:|---|---|---|---|
| Lv.1-10 | 白石渡残梦·天机 | 唐灵枢带我闯祸,鲁照衡修书扣 | 机关木偶、失控铜鸢 | 筑基资格 |
| Lv.11-25 | 沧澜《剑冢林泽案》 | 证明天机旧档不是伪造,林泽另有隐情 | 剑冢残剑、墨剑旧影 | 沧澜盟印 |
| Lv.26-45 | 玄机《无字天衍图》 | 天机与玄机必须共管天衍图 | 幻方阵灵、双算机关魁 | 玄机盟印 |
| Lv.46-70 | 青冥《青澜古碑注疏》 | 修复古碑注疏,洗清篡经嫌疑 | 纸灵、伪注碑灵 | 青冥盟印 |
| Lv.71-100 | 四宗归盟台 | 四宗各持一钥,玩家持总印 | 裂简伪盟、机关心魔 | 机关盟约/四宗重盟 |

小说化高潮:

```text
天机最痛的不是被冤枉,而是它真的藏过档案。
唐灵枢愿意陪我偷档案,但她也怕我把天机推到众矢之的。
当我在沧澜剑冢公开旧档时,她第一次不再开玩笑。
她问我: “如果真相会烧到我们自己,你还要开锁吗?”
```

### 3.3 玄机路线: 从不可推演到不可独占

主线情绪:

```text
我能破局,但我不能把活人都当棋子。
```

| 等级段 | 副本/章节 | 主线内容 | 打怪目标 | 卡点奖励 |
|---:|---|---|---|---|
| Lv.1-10 | 白石渡残梦·玄机 | 韩极简成为理性好友,周虚空若即若离 | 幻方残灵、观星影 | 筑基资格 |
| Lv.11-25 | 天机《双算天衍局》 | 处理韩极简与冯照矩的跨宗恋 | 机关棋盘、双算机关魁 | 天机盟印 |
| Lv.26-45 | 沧澜《三日不报旧案》 | 面对周虚空晚报三日的旧债 | 问剑台、迟报剑痕 | 沧澜盟印 |
| Lv.46-70 | 月隐《名册逆推》 | 证明记忆与推演都可能被篡改 | 记忆影卫、逆推名册 | 月隐盟印 |
| Lv.71-100 | 四宗归盟台 | 制定不能被任何一宗独占的盟约 | 裂简伪盟、推演心魔 | 暗盟共治/四宗重盟 |

小说化高潮:

```text
周虚空告诉我,当年他晚报三日是为了让我活下来。
陈墨痕不接受这个解释,墨听潮更不接受。
我必须决定: 是替师门承担罪名,还是逼周虚空公开全部真相。
这一刻,玄机路线从“我很聪明”变成“我愿不愿意为聪明的代价负责”。
```

### 3.4 青冥路线: 从藏书楼缺页到礼法归盟

主线情绪:

```text
我相信证据。
但如果证据证明本门错了,我也要写下去。
```

| 等级段 | 副本/章节 | 主线内容 | 打怪目标 | 卡点奖励 |
|---:|---|---|---|---|
| Lv.1-10 | 白石渡残梦·青冥 | 石听雨救我,文问松教我读经 | 纸灵、药田妖兽 | 筑基资格 |
| Lv.11-25 | 沧澜《青澜古碑护碑》 | 两派共守古碑,却都没守住真相 | 护碑剑灵、篡碑剑灵 | 沧澜盟印 |
| Lv.26-45 | 天机《机关锁与旧藏书楼》 | 查明旧锁曾打开藏书楼后门 | 锁库人偶、焚书偶 | 天机盟印 |
| Lv.46-70 | 月隐《书火翻案》 | 为月隐撤销百年旧判 | 月影审讯阵、书火残灯 | 月隐盟印 |
| Lv.71-100 | 四宗归盟台 | 用证据和礼法立下新盟 | 裂简伪盟、礼法心魔 | 礼法盟约/四宗重盟 |

小说化高潮:

```text
石听雨问我: “若真凶是青冥的人,你还会查吗?”
我若回答会,她会难过,但会继续给我送药。
这条线最抓人的地方,不是谁赢了,而是温柔的人也必须面对真相。
```

### 3.5 月隐路线: 从被藏起来的人到暗线立盟

主线情绪:

```text
我不是影子。
我要让四宗承认,有些秘密不是背叛,而是守护。
```

| 等级段 | 副本/章节 | 主线内容 | 打怪目标 | 卡点奖励 |
|---:|---|---|---|---|
| Lv.1-10 | 白石渡残梦·月隐 | 叶照眠暗护我,顾长忆修复记忆 | 月影残灵、封忆灯 | 筑基资格 |
| Lv.11-25 | 沧澜《掌刑堂赎罪线》 | 迫使沧澜面对误判月隐的旧错 | 掌刑剑阵、掌刑旧剑 | 沧澜盟印 |
| Lv.26-45 | 青冥《书火真凶线》 | 找到书火案真正证词 | 守书灵、焚卷真影 | 青冥盟印 |
| Lv.46-70 | 天机《名册机关锁》 | 重修名册机关锁,决定钥匙归属 | 锁库影卫、万同名册锁 | 天机盟印 |
| Lv.71-100 | 四宗归盟台 | 四宗共钥或暗盟共治 | 裂简伪盟、记忆心魔 | 暗盟共治/制度结局 |

小说化高潮:

```text
顾长忆承认,他曾亲手封过我的记忆。
叶照眠说,如果不封,我当年活不到现在。
沈鸦青却问: “被救的人,有没有权利恨救他的人?”
月隐路线的情感核心,就是救命恩和记忆伤害能不能同时成立。
```

## 4. 打怪如何变成主线需要

玩家必须不断打怪,但打怪不能只是刷数值。  
每个等级段怪物掉落都要服务主线副本。

### 4.1 掉落结构

| 掉落 | 来源 | 用途 |
|---|---|---|
| 有效修为 | 所有战斗 | 升级、破境 |
| 门派线索 | 目标门派地界怪物 | 解锁副本下一幕 |
| 旧案残证 | 精英怪/据点怪 | 开启问心节点 |
| 破境材料 | Boss/秘境/悬赏 | 通过等级卡点 |
| 关系信物 | 特定怪物/支线 | 推进固定羁绊 |
| 盟印共鸣 | 回响副本 | 解锁最终归盟台 |

### 4.2 怪物与副本绑定

| 玩家阶段 | 推荐怪物目标 | 叙事包装 |
|---|---|---|
| 炼气 | 山门试炼怪、低阶妖兽 | 清理本门地界,证明自己不是拖累 |
| 筑基 | 目标外宗地界怪 | 追查第一盟印线索 |
| 金丹 | 妖巢、机关失控点、书灵据点 | 旧案证据被妖物或心魔污染 |
| 元婴 | 高阶守关怪、门派影卫 | 目标门派不再让你随便查 |
| 化神 | 三宗回响、裂简伪影 | 过去选择化成心魔和天劫 |

### 4.3 副本节点解锁示例

以沧澜玩家的青冥副本为例:

```text
旧藏书楼书火案
```

| 节点 | 解锁要求 |
|---|---|
| 入境 | Lv.26 金丹完成 |
| 起疑 | 击败 5 只青冥地界书灵 |
| 试炼一 | 收集 3 个“焦黑经页” |
| 试炼二 | 击败精英怪“灰烬守卷灵” |
| 问心 | 石听雨好感 30 或文问松支线完成 |
| Boss | 修为达到本级 80%,持有“书火残证” |
| 归盟 | 击败守卷心魔,选择是否公开月隐无罪线索 |

玩家看到的是:

```text
我不是在刷 5 只怪。
我是为了凑齐焦黑经页,打开旧藏书楼下一层。
```

## 5. 主线副本必须有小说感

### 5.1 每个副本都要有一个情感问题

| 副本类型 | 情感问题 |
|---|---|
| 机关副本 | 真相如果会伤害本门,还要不要开锁 |
| 剑派副本 | 复仇能不能代替正义 |
| 经卷副本 | 证据如果证明自己错了,还要不要写 |
| 名册副本 | 救命恩和记忆伤害能不能同时成立 |
| 推演副本 | 为了大局牺牲一个人,算不算正确 |
| 最终副本 | 统一四宗会不会变成新的独占 |

### 5.2 每个破境都要有一个人物代价

| 破境 | 人物代价 |
|---|---|
| 筑基 | 初始熟人开始知道你不是普通弟子 |
| 金丹 | 第一个外宗盟友因你被门内怀疑 |
| 元婴 | 一个重要角色受伤、失踪或与你决裂 |
| 化神 | 三枚盟印让所有门派都要求你表态 |
| 合体以后 | 统一后的制度开始反噬个人关系 |

### 5.3 主线章节标题建议

每条路线可以像小说目录一样展示:

| 路线 | 章节名 |
|---|---|
| 沧澜 | 白石渡残梦、锁心问天、书火留痕、月册无名、剑誓归盟 |
| 天机 | 活钥入阁、剑冢旧档、无字双算、碑下注疏、四钥归一 |
| 玄机 | 空命入局、双算天衍、三日旧债、名册逆推、无局归盟 |
| 青冥 | 灰烬书童、古碑共誓、旧锁焚书、月下翻案、礼法立盟 |
| 月隐 | 名册藏名、掌刑赎罪、书火真凶、四钥重锁、暗盟共治 |

让玩家主线面板更像一本书:

```text
当前章节: 书火留痕 · 第三幕
下一目标: 击败灰烬守卷灵,取得书火残证
推荐等级: Lv.38
牵涉人物: 石听雨 / 文问松 / 陆正辞
```

## 6. 落地实现优先级

| 优先级 | 内容 |
|---|---|
| P0 | 在 `next_level_need` 对应 UI 中显示“等级卡点 + 主线副本条件” |
| P0 | 在玩家世界存档加入 `main_story_progress` |
| P0 | 每个门派路线绑定: Lv.10、25、45、70 的剧情条件 |
| P1 | 打怪掉落“门派线索/旧案残证/破境材料” |
| P1 | 副本节点按等级和线索逐步解锁 |
| P1 | 破境按钮检查: 修为 + 副本条件 + 材料 |
| P2 | 每次破境生成一章本命书正传 |

### 6.2 升级界面文案

不要只写:

```text
修为不足,无法升级。
```

应写:

```text
距离金丹尚缺:
修为 84,000
青冥盟印 未获得
书火残证 2/3

推荐:
前往旧藏书楼书火案 · 第三幕
击败灰烬守卷灵,可获得书火残证。
```

这样玩家知道自己为什么要去打怪。
